# 防身指南 05｜主治 vs PGY/R 責任分擔（指揮監督義務）

## 來源
- 醫師法 §28（密醫罪/業務範圍）
- 醫療法 §82（醫療責任）
- 最高法院 102 年度台上字第 3161 號（指揮監督義務經典）
- 最高法院 95 年度台上字第 3884 號（住院醫師獨立判斷）
- 信心等級：高

---

## 一、白話結論（30 秒）

**「PGY/R 出包，主治也要扛；但主治 100% 不知情的，PGY/R 自己負責。」**

關鍵不是「誰下決定」，而是 **「主治有沒有盡到指揮監督義務」**。

---

## 二、責任分配光譜

| 情境 | 主治責任 | PGY/R 責任 |
|---|---|---|
| 主治在場下指令 → PGY 執行 | **主要責任** | 輔助 |
| 主治值班但 PGY 沒 call → 出包 | 監督義務違反 | 主要責任 |
| 主治休假（未交班）→ PGY 自行決定 | 院方系統責任 | 仍在能力範圍內 |
| PGY 超出能力範圍 + 沒求助 → 出包 | 部分監督責任 | **重大過失** |
| 急救 PGY 第一線到場 → 主治後到 | 微 | PGY 救治義務 |

---

## 三、最高法院 102 台上 3161 號（必讀）

**事實**：實習醫師執行某 procedure 出錯，主治不在場。
**爭點**：主治是否有監督義務違反。
**判決**：「**指導醫師對於實習醫師之指導及監督義務，應依其資歷與能力差異程度而定**…未盡監督義務者，與實習醫師同負過失責任。」

→ 翻譯：**PGY 越資淺，主治監督義務越重**。

---

## 四、PGY/R 自保 5 條

### 1️⃣ 「我不會」要說出口
- 自己沒做過的 procedure → 直接說 **「請學長/主治在旁監督」**
- 不要「先試試看」 — 出事是「重大過失」
- 法源：醫師法 §28-4「執行業務逾越其能力範圍」

### 2️⃣ Stat order 親自確認
- 開了急救藥 → 親自看到 / 聽到「已給」才能離開
- 護理人力不足時更要 read back

### 3️⃣ 半夜 call 主治不要怕
- 「半夜不敢吵主治」 = **法律上不是抗辯理由**
- 病歷紀錄寫「已電話請示 OOO 主治，回覆 XXX」 = 主治分擔責任
- **沒 call 出包 = PGY 全扛**

### 4️⃣ 病歷上把指令來源寫清楚
- ✅ 「依 OOO 主治指示給 Lasix 40mg IV」
- ❌ 「給 Lasix 40mg IV」（誰下的？沒人知道）
- 將來訴訟 → 病歷沒寫指令來源 = PGY 變主謀

### 5️⃣ 交班一定要書面
- 口頭交班 = 出事查不到
- LINE 群組交班 = 個資疑慮（可截圖但非正規）
- 三總用 eForm 交班 → 留底

---

## 五、主治也別誤會：你也有自保義務

- 接到 PGY 半夜 call → **不要一句「自己看著辦」**
  - 出事鑑定會看「主治是否提供具體指示」
- 「沒 call 我」抗辯不一定成立
  - 法院會問「你的監督機制是什麼？」
  - 沒交班、沒清單 → 院方系統責任 + 主治監督失靈

---

## 六、實務真實案例

| 案例 | 結果 |
|---|---|
| 實習醫師抽脊髓液漏看顱壓升高（最高法院 102 台上 3161） | 指導醫師同負過失 |
| R1 急診胸痛漏 STEMI、主治值班未到場 | 主治+R1 連帶責任 |
| PGY 自行 discharge 闌尾炎漏診（類似 Case 03） | 後繼義務未盡，主治+PGY 同負 |

## 七、引用法條
- 醫師法 §28（業務範圍）、§28-4（逾越能力）、§21（急救義務）
- 醫療法 §82（醫師個人責任 = 違反義務 + 逾越裁量）
- 民法 §188（僱用人連帶責任）— 醫院仍負連帶
- 刑法 §276（過失致死）

## 八、口袋提醒
> 出事時法院問的不是「你會不會」，而是 **「你有沒有開口求助 + 病歷有沒有寫」**。
