# 各科地雷 #4：兒科（社會同情效應最強）

> 兒科訴訟雖數量不及外科，但**社會同情效應**讓賠償金額不容小覷。
> 5 個最常被告的情境 + 防身做法。

---

## 為什麼兒科訴訟特別棘手

| 因素 | 影響 |
|------|------|
| 病人不會自述 | 漏診風險 ↑ |
| 兒童無法自己決定 | 同意書由父母簽，意見衝突常見 |
| 家屬情緒高漲 | 不接受任何 bad outcome |
| 終身殘疾賠償計算 = 年數 × 平均薪資 | 金額容易破千萬 |

## 🧒 雷區 1：嬰幼兒 sepsis / meningitis 漏診

### 法律邏輯
腦膜炎漏診 → 永久神經損傷 → 賠償極高

### PGY 防身做法

**< 3 個月嬰兒發燒處理 SOP**：
1. **必抽血**：CBC + CRP + BC（即使外觀看起來 OK）
2. **強烈考慮 LP**（< 1 個月幾乎是必做）
3. **強烈考慮 admission**
4. **不要因家屬「他活力很好」就放走**

**Toxic appearance signs**（任一即警戒）：
- Lethargy / poor feeding
- Bulging fontanelle
- Mottled skin / cold extremities
- Capillary refill > 2 sec
- Petechiae / purpura（meningococcemia！）

**meningitis triad**（兒童不完整表現很常見）：
- Fever ✓
- Headache（嬰兒不會說，看 irritability）
- Neck stiffness（嬰兒 fontanelle 鼓起）

→ **任一懷疑 → LP**

### 病歷救命寫法
```
2 月大男嬰，發燒 39.0°C 12 hr，活力稍降，吃奶量減少
PE: anterior fontanelle slight bulging, no rash
   irritable when handled, otherwise vital sign stable
A: < 3 個月 fever, r/o sepsis / meningitis（不能排除）
P: 1. CBC + CRP + BC + UA + UC
   2. LP 已向父母說明風險與必要性，父母同意（簽名）
   3. Empirical antibiotics（ampicillin + cefotaxime）after culture
   4. Admit
**已詳細向父母說明：嬰兒發燒下能引起腦膜炎，即使活力尚可仍須完整評估**
```

## 🧒 雷區 2：腸套疊（intussusception）漏診

### 法律邏輯
- 6 月-3 歲是高峰
- **延誤 > 24 hr** = 腸壞死、敗血症
- 「典型三聯徵」（陣發腹痛、嘔吐、血便/果醬便）只在 1/3 病人完整出現

### PGY 防身做法

**警戒情境**：
- 6m-3y 兒童陣發腹痛 + 嘔吐
- 「沒精神」「腿縮起來哭」（陣發疼痛時）
- 觸診 sausage-shaped mass（RUQ）
- 血便 / 果醬便 = late sign（已壞死）

**處置**：
1. 腹部超音波（target sign / pseudokidney sign）
2. **air enema or barium enema** 復位（成功率 80-90%）
3. 失敗 → 急診手術

→ **即使沒有完整三聯徵也不排除**，超音波必做

## 🧒 雷區 3：兒童虐待 / 非意外傷害（NAI）

### 法律雙刃劍
- **沒通報** = 違反兒童及少年福利與權益保障法 §53（強制通報）
- **錯誤通報** = 可能被告誣告（但實務上甚少）
- **法律保護**：依法通報的醫師不負民刑事責任（§53 IV）

### PGY 防身做法

**警戒徵兆**：
- 受傷與病史不符
- 多處不同階段瘀青
- 燙傷邊界整齊（dipping pattern）
- 父母解釋不合邏輯
- 延誤就醫
- 重複受傷史

**通報流程**：
1. 24 小時內通報衛福部 113 專線或當地社政
2. 病歷詳細記錄：傷勢、父母陳述、不合理處
3. 拍照存證（依院方 SOP）
4. **通報是法定義務**，不需父母同意

## 🧒 雷區 4：未成年同意權問題

### 法律核心
- 醫療法 §63 II：「病人為未成年人或無法親自簽具者，得由其法定代理人、配偶、親屬或關係人簽具」
- **法定代理人 = 父母**（婚生子女）

### 棘手情境

#### 情境 A：父母離婚，由誰簽？
- 看「**單獨監護權 or 共同監護權**」
- 共同監護權 → 重大手術建議**雙方都同意**
- 單獨監護權 → 監護權人簽即可
- 緊急手術可不等

#### 情境 B：父母意見不一致
- 一方堅持要做、一方堅持不做
- 建議：**啟動院方倫理委員會 + 社工介入**
- 不要單一醫師決定

#### 情境 C：青少年 vs 父母
- 衛福部解釋令：**未成年人對自己的醫療有發言權**（依年齡與成熟度）
- 16 歲以上青少年 + 簡單處置 = 可由本人同意（如門診處方）
- 重大手術仍由法定代理人

#### 情境 D：父母拒絕醫學上必要的治療
- 例：耶和華見證人拒絕輸血給孩子
- **緊急情況**：醫師可依醫療必要性處置（醫師法 §21）
- **非緊急**：尋求法院裁定（社工協助）

## 🧒 雷區 5：藥物劑量錯誤

### 法律邏輯
兒童藥物依體重計算（mg/kg），算錯一個小數點 = 災難

### PGY 防身做法

**雙重檢查 SOP**：
1. 自己算一遍 + 護理師算一遍
2. 體重以 dosing weight 計算（特殊：肥胖、低體重）
3. **危險藥物**（chemo、insulin、heparin、digoxin）→ 三人 check
4. 高警示藥品（high-alert medications）依院方 SOP

**最常算錯的場景**：
- 急診情境壓力下
- 體重單位混淆（kg vs lb）
- 兒童 vs 成人劑量單位差異（mg/kg vs mg）
- 半 / 1/4 / 1/8 顆藥的分割

### 病歷救命寫法
```
3 y/o 男童，10.5 kg，急性中耳炎
Amoxicillin 90 mg/kg/day, divided q12h
= 90 × 10.5 = 945 mg/day
= 472.5 mg q12h ≈ 500 mg q12h
（已雙重檢查，與護理師 王XX 共同確認）
```

## 兒科同意書特別注意

**家屬簽名順序**（醫療法 §63 II）：
1. 法定代理人（父母）
2. 監護人（如父母不在）
3. 配偶（婚生未成年配偶）
4. 親屬（祖父母、成年兄姊）
5. 關係人

**特殊**：
- 棄嬰、無人領養 → 社政機構為法定代理人
- 緊急手術 + 無法立刻聯絡家屬 → 醫師法 §21 緊急處置 + 事後補簽
