## 👶 雷區 1：胎心音監測判讀失誤

### 真實案例對應
- **case06 北榮胎心音漏判腦麻**（醫師未親自判讀 CTG → 1905 萬）

### 楊坤仁大師的核心觀點
「醫師不能只『接上監測儀器』，必須親自判讀 + 記錄判讀結果」（master_yang_kun_ren.md 觀點二）

### PGY 防身做法

**CTG 判讀記錄要點**：
| 項目 | 怎麼寫 |
|------|--------|
| Baseline | bpm 範圍（normal 110-160）|
| Variability | absent / minimal / moderate / marked |
| Accelerations | present / absent |
| Decelerations | early / late / variable / prolonged |
| Category | I / II / III（NICHD 分類）|

**Category III（need immediate action）**：
- Recurrent late decelerations + minimal variability
- Sinusoidal pattern
- Bradycardia + minimal variability

→ 看到 Category III → **立刻處置 + 記錄時間 + 通知主治**

### 病歷救命寫法
```
14:30 CTG: baseline 145 bpm, moderate variability,
       2 accelerations in last 20 min, no decelerations → Cat I
14:45 仍 Cat I, NICHD 標準
15:30 出現 late deceleration × 2，恢復 baseline
       → Cat II, 已給左側臥 + O2 4L NC + IV fluid
       → 通知主治 王 XX，於 15:35 評估
15:45 持續 Cat II，繼續 conservative management
       → 16:00 重新評估，如 Cat III 立即 emergent CS
```

## 👶 雷區 2：肩難產處理不當

### 真實案例對應
- 肩難產（shoulder dystocia）是婦產科最戲劇性的緊急事件
- 文獻顯示為 brachial plexus injury 的主要原因
- 訴訟金額可達數百萬至上千萬

### PGY 防身做法（HELPERR mnemonic）

H - **H**elp（call 額外人手：另一位產科、麻醉、新生兒科）
E - **E**valuate for episiotomy
L - **L**egs（McRoberts maneuver：媽媽腳折向腹）
P - **P**ressure（suprapubic pressure，**不要 fundal pressure**）
E - **E**nter（rotational maneuvers：Rubin / Woods）
R - **R**emove posterior arm
R - **R**oll（Gaskin maneuver：媽媽四肢著地）

**處置時間軸要寫入病歷**（每一步什麼時候做）：
```
14:32:15 fetal head delivered
14:32:30 shoulder dystocia 確認，call extra help
14:32:45 McRoberts + suprapubic pressure
14:33:10 Rubin maneuver attempt
14:33:30 successful delivery
APGAR 1' = 5, 5' = 8
```

→ **時間戳記精確到秒**，這是肩難產防身的關鍵。

## 👶 雷區 3：催生時機與方式選擇

### 真實案例對應
- **case05 大里仁愛真空吸引致死**（羊水胎便染色 + 真空吸引使用時機 → 1404 萬）

### 法官會問什麼
1. 「真空吸引使用適應症是什麼？」
2. 「Pop-off 幾次了？」
3. 「為什麼不轉 CS？」
4. 「胎便染色為什麼還繼續真空吸引？」

### PGY 防身做法

**真空吸引適應症（嚴格遵守）**：
- 第二產程延長
- 母體用力效能不足（如心臟病）
- 胎兒心率異常需立即分娩

**真空吸引限制**：
- **3-3-3 原則**：3 次 pop-off / 3 次拉動無下降 / 3 次 cup application
- **超過 = 改 CS 或 forceps**
- 胎兒 < 34 週 = 禁用

**胎便染色（meconium staining）處理**：
- 1-2 成生產有胎便，**不一定要立刻 CS**（婦產科醫學會立場）
- 但要**密切監測 CTG**：胎便 + Category II/III → 考慮 CS
- 病歷記錄評估過程：「meconium staining noted, but CTG remains Cat I, continue close monitoring」

## 👶 雷區 4：產後出血應變

### 法律邏輯
產後出血（PPH）是孕婦死亡主因之一。延誤 → 民事敗訴幾乎必然。

### PGY 防身做法

**4T 鑑別**：
- **T**one - 子宮收縮不良（最常見）
- **T**rauma - 產道撕裂
- **T**issue - 胎盤殘留
- **T**hrombin - 凝血異常

**Stage approach**（依出血量）：
| Stage | 出血量 | 處置 |
|-------|--------|------|
| 1 | 500-1000 mL | massage、oxytocin、check vital signs |
| 2 | 1000-1500 mL | call extra help、活化 PPH protocol、type & cross |
| 3 | > 1500 mL | massive transfusion protocol（MTP）、surgical/interventional |
| 4 | > 2500 mL | hysterectomy 考慮、ICU |

**病歷必寫**（每 stage）：
- 時間戳記
- 出血量估計（明確說數字，不要寫「大量」）
- Vital sign
- 處置內容
- 通知誰

## 👶 雷區 5：手術同意書（剖腹產 / 子宮切除）

### 真實案例對應
- **case09 北聯醫子宮肌瘤切除致植物人**（4000 萬）
- **case02 新竹馬偕腹腔鏡卵巢畸胎瘤**（2198 萬）

### PGY 防身做法

**剖腹產同意書必寫項目**：
- 適應症（CPD、breech、fetal distress、prior CS）
- **替代方案**（VBAC、繼續嘗試陰道生產）
- 主要併發症：
  - 出血需輸血（1-2%）
  - 子宮切除（< 1%）
  - 周邊器官損傷（膀胱、腸道）
  - 感染（< 5%）
  - 麻醉相關
  - 對未來生產影響
- **再次手術可能性**

**子宮切除前**：
- 必須充分說明對「**生育功能**」的永久影響
- 替代方案（保守治療、子宮動脈栓塞、保留子宮的手術）
- 卵巢留不留的決策

## 婦產科特殊：生產事故救濟條例（2016）

### 制度核心
- 政府提供「無過失補償」
- 不需證明醫師過失
- 補償項目：母體死亡、嚴重併發症；新生兒死亡、重度障礙

### 法律意義
- **降低訴訟壓力**：家屬拿到補償後，提告意願下降
- **不影響訴訟權**：家屬仍可提告，但實證上減少
- **吳俊穎大師參與推動**（master_wu_jun_ying.md 觀點三）

### PGY 實務啟示
- 發生事故 → **第一時間告知家屬此制度**
- 申請電話：02-2100-2092
- 醫院通常有專人協助申請
