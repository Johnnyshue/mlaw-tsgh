# 各科地雷 09｜心臟內科（STEMI 黃金時間 + ECMO 決策）

## 來源
- 醫療法 §82
- 案例佐證：Case 11（葉克膜延遲案，無罪）
- AHA/ESC Guidelines、台灣心臟學會
- 信心等級：高

---

## 一、白話結論

**「胸痛 EKG 沒做」 + 「STEMI 等檢驗報告才送 cath」 = 心內最常被告兩條路。**

---

## 二、5 大訴訟高風險情境

### 1️⃣ ACS 沒做 EKG
- 醫療常規：**胸痛 10 分鐘內 EKG**
- > 30 歲胸痛 → EKG + Troponin 是底線
- 沒做 → 漏診 STEMI / NSTEMI → 民事 / 刑事
- 案例：多起急診胸痛漏診 STEMI 賠償

### 2️⃣ STEMI 黃金時間錯過
| 項目 | 標準 |
|---|---|
| Door-to-EKG | ≤ 10 分 |
| Door-to-Balloon (D2B) | ≤ 90 分 |
| Door-to-Needle (溶栓) | ≤ 30 分 |

- 等檢驗報告再決定 → 通常已超時
- **Time is muscle** → 每 30 分延誤 = 心肌死更多

### 3️⃣ 主動脈剝離漏診（誤當 ACS 給雙抗 → 出血死亡）
- 突發劇烈撕裂痛、血壓不對稱、縱膈腔擴大 → 警示
- 給 antiplatelet + heparin → 致死併發症
- 病歷必寫：是否考慮過 dissection、CT aorta indication 評估

### 4️⃣ 心衰用藥地雷
- HFrEF 必備 4 神器（ACEi/ARNI、beta-blocker、MRA、SGLT2i）沒給 → 醫療常規違反
- 病歷必寫禁忌症（hyperkalemia、HoTN）
- 利尿劑過度 → AKI → 民事

### 5️⃣ ECMO / IABP 決策延遲（Case 11）
- Case 11（最高法院 95 台上 4476、後續無罪）：葉克膜延遲被起訴 → 一二審有罪 → 三審無罪
- **重點**：醫療裁量 + 工作條件 → 不必過度設防衛醫療
- 但**病歷必寫評估理由 + 風險告知**

## 三、心內常見鑑定爭點

| 爭點 | 鑑定重點 |
|---|---|
| 該不該送 cath lab | EKG 結果、症狀、時間 |
| 該不該打 dual antiplatelet | 出血風險、ACS 確診 |
| 該不該插 ECMO | Cardiogenic shock 程度、可逆性、家屬意見 |
| 該不該放 ICD/CRT | LVEF、症狀、indication |

---

## 四、PGY 心內自保 5 條

### 1️⃣ EKG 第一張 → 直接掃描存檔
- 病歷貼上時間戳
- 連續 EKG 變化（每 15 分追蹤）
- 沒對照 → 鑑定沒依據

### 2️⃣ Stat order 親自確認
- Aspirin 300mg loading、Heparin、急救藥
- 護理人手不足時尤其重要

### 3️⃣ 家屬告知必有書面
- STEMI 風險、cath 風險、ECMO 風險
- 拒絕治療簽 AAD + 詳實紀錄

### 4️⃣ 心律不整不要僅靠口頭交班
- VT/VF → 立刻 ACLS
- 用 LINE 通知主治 → 也要病歷紀錄

### 5️⃣ Discharge 衛教 + 用藥順從
- ACS post-discharge：DAPT 至少 12 個月
- 衛教：紅旗症狀（胸痛回診、出血）
- 病歷寫衛教內容 + 簽收

---

## 五、ECMO 法律觀點（Case 11 啟示）

- 醫療法 §82 修正後：醫師個人責任 = **違反義務 + 逾越裁量**
- ECMO 設置條件：人力、設備、可逆性
- 醫師裁量範圍內的決定（即使結果不好）→ 不應推定有過失
- 但**病歷必須寫**：考量、家屬討論、決定理由
