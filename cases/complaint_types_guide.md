# 申訴類型圖解：病人/家屬有 N 條路告你，你要懂每一條

> 「告你」不只是「上法院」。
> 病人/家屬有**至少 7 種**正式管道對你發動，每種戰略不同。
> 你懂得越多，反應越好，傷害越小。

---

## 七大申訴管道一覽

| # | 管道 | 處理機關 | 時間 | 後果嚴重度 |
|---|------|---------|------|-----------|
| 1 | **院內申訴** | 醫院申訴專線 / 病人權益室 | 1-4 週 | 🟢 低（多數在這停下） |
| 2 | **衛生局申訴** | 縣市衛生局 | 2-3 個月 | 🟡 中 |
| 3 | **醫預法調解** | 縣市調解會 | 3-6 個月 | 🟡 中 |
| 4 | **健保署申訴** | 中央健保署 | 1-3 個月 | 🟡 中（限健保問題）|
| 5 | **媒體申訴 / 投書** | 媒體 / 議員 | 即時 | 🔴 高（社會壓力）|
| 6 | **民事訴訟** | 地方法院民事庭 | 1-3 年（一審）| 🔴 高（賠錢）|
| 7 | **刑事告訴** | 地檢署 | 6 個月-2 年（偵查）| 🔴🔴 極高（坐牢）|

---

## 1. 院內申訴（最常見，多數在此停下）

### 觸發
- 家屬向護理長、醫院投訴專線抱怨
- 顧客滿意度問卷負評
- 醫院信箱

### 處理流程
```
家屬投訴 → 病人權益室收件
  ↓
調閱病歷 + 詢問醫師
  ↓
回覆家屬（口頭 + 書面）
  ↓
家屬接受 → 結案
家屬不接受 → 升級（衛生局或調解）
```

### 對醫師的影響
- 通常**不留紀錄**（除非醫院內部評定有疏失）
- 主治醫師可能找你談話
- 嚴重者有「事件報告」（內部）

### 你該做的
1. 配合院方說明
2. 提供完整病歷
3. **不要直接聯繫家屬**（讓院方統一說明）
4. 寫個人時間軸

---

## 2. 衛生局申訴（升一級）

### 觸發
- 院內申訴未解決
- 家屬直接向衛生局申訴

### 處理流程
```
家屬申訴 → 衛生局收件
  ↓
衛生局派員調查（書面 + 訪談）
  ↓
醫療機構提供資料
  ↓
衛生局召開審議
  ↓
結果：駁回 / 行政處分（罰鍰、停業）
```

### 可能行政處分（醫療法 §102~§115）
- 罰鍰（5 萬-25 萬）
- 限期改善
- 停業（1 個月-6 個月）
- 廢止執照（嚴重時）

### 對醫師個人的影響
- **個人有紀錄**（衛生局留檔）
- 影響升等、評鑑
- 嚴重者影響執照

### 你該做的
1. **配合調查**（不能拒絕）
2. **找律師陪同**訪談
3. 通知保險公司（雖然行政處分多不在保險範圍，但律師費可能含）

---

## 3. 醫預法調解（2024 年新法後強制）

### 觸發
- 家屬要提民事訴訟前 = **必須先調解**
- 醫療機構也可主動申請

### 處理流程
```
申請調解 → 縣市調解會
  ↓
組成調解小組（醫師 + 律師 + 公正人士）
  ↓
雙方陳述 + 鑑定
  ↓
調解成立 → 簽和解書 → 法院核定
調解不成 → 出具不成立證明 → 才能起訴
```

### 醫預法 §7 道歉條款
- 調解中說的「**遺憾、道歉、讓步**」
- 除雙方同意外，**不得用於訴訟**
- 你可以放心表達遺憾

### 醫預法 §15-16 程序保護
- 民事訴訟前**強制調解**
- 刑事偵查/審理在調解期間**暫停**

### 對醫師的好處
- 可以「不經訴訟」結案
- 賠償金額通常低於訴訟判決
- 不留刑民事紀錄

### 你該做的
1. **積極配合調解**（不要拒絕）
2. 通知保險公司（很多保單覆蓋調解費用）
3. 律師陪同
4. 表達遺憾（受 §7 保護）

---

## 4. 健保署申訴

### 觸發
- 限**健保相關**問題
- 例：浮報、虛報、自費問題、藥物給付爭議

### 處理流程
- 健保署收件 → 調查 → 行政處分
- 違規可罰：扣扣費 / 暫停申報 / 廢止合約

### 對醫師個人
- 申報違規 = 健保扣費 + 可能觸發醫師法 §28
- 嚴重：刑事（詐欺）

### 你該做的
1. 確認申報合規
2. 諮詢科內藥師 / 主任
3. 律師（如涉刑事）

---

## 5. 媒體申訴 / 投書

### 觸發
- 家屬覺得院方推託
- 議員介入
- 上社會新聞

### 對醫師的影響
- **社會壓力極大**
- 訴訟心證可能受影響
- 個人形象受損
- 醫院/科室評鑑受影響

### 你該做的
1. **不要單獨對媒體發言**（看 media_response_guide.md）
2. 通知院方公關
3. 諮詢律師（是否提妨害名譽）
4. 不要在社群媒體反駁

---

## 6. 民事訴訟

### 觸發
- 家屬決心追求賠償
- 醫預法調解不成
- 已有刑事告訴在進行（民事跟著告，效率較高）

### 流程
```
起訴 → 一審地方法院（1-3 年）
       ↓
       醫審會鑑定（6-18 個月）
       ↓
       判決（賠 / 不賠 / 部分賠）
       ↓
二審高等法院（1-2 年）
       ↓
三審最高法院（< 1 年）
```

### 對醫師個人
- 賠償金額：80 萬中位數，最高至 4000 萬
- 訴訟費：律師費 50-100 萬起
- 時間成本：3-10 年
- 精神壓力：極大

### 你該做的（按時序）
1. **接到起訴狀第一時間** → 通知保險、律師、法務
2. **30 天內**書面答辯
3. **配合鑑定**（提供病歷、書面說明）
4. **準備證據**（guideline、文獻、證人）
5. **應訊技巧**（看 sop_when_sued.md）

---

## 7. 刑事告訴

### 觸發
- 病人死亡 / 重傷
- 家屬要求醫師「**坐牢**」
- 過失致死非告訴乃論 → **檢察官可主動偵查**
- 改死因 / 修病歷 → 自動觸發

### 流程
```
告訴狀 → 地檢署
       ↓
       檢察官偵查（傳訊、調查、鑑定）（6-18 個月）
       ↓
       起訴 / 不起訴 / 緩起訴
              ↓
       一審地方法院刑事庭（1-2 年）
              ↓
       二審 / 三審
```

### 結果可能性（依文獻）
- 不起訴 / 無罪：> 60%
- 緩起訴 / 緩刑：~ 20%
- 有罪入監 / 易科罰金：~ 20%（多為輕罪）

### 對醫師個人
- 留前科（即使易科罰金）
- 影響升等
- 嚴重者廢執照（醫師法 §28）
- **執業生涯可能結束**

### 你該做的
1. **立刻找專業刑事律師**（醫療法律經驗者）
2. 通知保險（多數保單刑事辯護費覆蓋）
3. 不要對檢察官「自願多說」
4. 行使緘默權的智慧（法定權利）
5. 配合鑑定但**仔細閱讀鑑定報告**

## 戰略：不同管道的「軟硬度」

```
🟢 軟（多溝通可解決）
   院內申訴 → 衛生局申訴 → 健保申訴

🟡 中（要走程序）
   醫預法調解

🔴 硬（要打訴訟戰）
   民事 → 刑事
```

**重點**：你**最希望停在「軟」的層級**。
- 第一時間誠實面對 + 配合院方流程 = 70% 案件停在這
- 升級到調解 = 20%
- 升級到訴訟 = < 10%

→ 「越早處理 = 越輕」這個原則永遠對。

## 多條路同時走怎麼辦

### 真實情境
家屬可能同時：
- 院內投訴（第 1 天）
- 衛生局申訴（第 1 週）
- 媒體投書（第 2 週）
- 民事起訴（第 3 個月）
- 刑事告訴（第 4 個月）

### 你的應對策略
1. **每條都認真處理**，不能放棄
2. **同一律師團隊處理**（避免說法不一致）
3. **時序記錄完整**（哪個案件什麼狀態）
4. 保險公司**全程通知**

---

## 結論：保命策略

1. **病歷紀錄好** → 多數申訴在「院內」就解決
2. **個人責任險** → 萬一升級到訴訟，財務有底
3. **接到通知 24 小時內**：保險公司 + 律師 + 院方法務
4. **不修改病歷、不對媒體發言、不私下和解**
5. **配合院方關懷小組**：醫預法新時代下這是合法且有效的緩衝
