# 法律分析 02 | 民事醫療過失：舉證責任與轉換原則

## 來源
- 主要來源：https://www.legis-pedia.com/article/health-medical-ageing/1183
- 最高法院 106 年度台上字第 227 號
- 信心等級：高

---

## 一、基本原則

民事訴訟法 §277：**當事人應就有利於己之事實負舉證責任。**
原則上 → **原告（患者/家屬）舉證醫師有過失及因果關係**

---

## 二、舉證責任轉換的三種情況

### 情況 1：重大醫療瑕疵（最高法院 106 台上 227）
條件：
1. 患者能證明存在重大醫療瑕疵
2. 瑕疵造成因果關係難以釐清

效果：**舉證責任轉換，改由醫師舉證「其過失行為與損害間無因果關係」**

### 情況 2：病歷記載不完整
台大急診頭部外傷案：「病歷中全無昏迷指數及生命徵象記載 → 重大醫療瑕疵 → 因果關係不明 → 醫師負舉證責任。」

### 情況 3：病歷銷毀/竄改
「基於公平衡量，依舉證責任轉換原則，由醫師舉證無因果關係。」

## 三、賠償項目與統計

### 賠償項目
| 項目 | 說明 |
|------|------|
| 醫療費 | 已發生及未來必要費用 |
| 看護費 | 終身看護（植物人案件金額最大）|
| 勞動能力損失 | 餘命 × 所得 × 中霍夫曼係數 |
| 精神慰撫金 | 患者本人 + 家屬 |
| 喪葬費（死亡）| 實際費用 |

### 統計（2016–2019，doctor119.tw）
| 指標 | 數值 |
|------|------|
| 病家勝訴率（一審）| 11% |
| 平均判賠 | 157 萬 |
| 中位數 | 80 萬 |
| 2021 年平均 | 約 300 萬（顯著上升）|

## 四、醫院連帶責任（民法 §188）

「受僱人執行職務不法侵害他人，由**僱用人與行為人連帶負賠償責任**。」
→ 患者通常同時告醫師 + 醫院
→ 醫院連帶 → 求償較有保障
