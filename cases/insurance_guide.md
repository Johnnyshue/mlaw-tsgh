# 保險與救濟全攻略：醫師最容易忽略的「第二個防線」

> 病歷 + 同意書是第一防線。保險 + 救濟制度是**第二防線**。
> 萬一第一防線破了，第二防線決定你是「賠到破產」還是「全身而退」。

---

## 一、為什麼必須懂保險

### 統計感覺
- 醫療民事平均判賠 157 萬（中位數 80 萬）
- 2021 年平均上升至約 300 萬
- 高風險科：婦產 1404-4000 萬、骨科常見 200-500 萬
- **訴訟期間 3-10 年**，律師費 + 訴訟費 50-100 萬起跳

### 醫師個人責任險的核心保護
- 訴訟費（律師費 + 法院費用）
- 賠償金（按保額上限）
- 鑑定費
- 部分含「形象修復」（媒體危機處理）

→ 沒保險 = 全部自己扛，可能影響 5-10 年職涯規劃。

---

## 二、四大救濟制度（不一定要走訴訟）

### 1. 醫療事故預防及爭議處理法調解（醫預法，2024）
- 民事訴訟前**強制先調解**
- 醫師、律師、社會公正人士組成調解會
- 調解書經法院核定 = 不得再提同一事件民/刑事
- **為醫師爭取的時間**：刑事偵查/審理會暫停

### 2. 生產事故救濟條例（2016）
- 婦產科專屬
- 母嬰生產相關死亡 / 重度殘疾
- **無過失補償**（不需證明醫師過失）
- 補償上限 200 萬（依條例）
- 申請電話：02-2100-2092
- 不影響訴訟權，但實證減少訴訟率

### 3. 藥害救濟基金（藥害救濟法）
- 因「**正當使用合法藥物**」發生嚴重不良反應
- 死亡、障礙、嚴重疾病
- 基金來源：藥廠繳納
- 申請：衛福部 食藥署 藥害救濟基金會
- 對醫師意義：**減少病人對醫師的求償壓力**

### 4. 預防接種受害救濟（傳染病防治法）
- 因預防接種發生死亡、障礙、嚴重疾病
- 衛福部疾管署
- 申請流程約 6-12 個月

## 三、醫師個人責任險（你必懂的細節）

### 保費粗估（依科別差異大）
- 內科 / 家醫 / 復健：較低
- 外科 / 婦產 / 麻醉 / 急診：較高
- 醫美 / 整形：最高
- PGY 階段：通常較便宜（甚至有公會團保）

### 投保管道
1. **醫師公會團保**：基本保障最便宜
2. **產險公司個人險**（富邦、國泰、新光等）
3. **責任險 + 名譽險組合**：有些保單含媒體危機

### 投保前必問 5 件事
1. 保額上限（單一事件、年度上限）多少？
2. 訴訟費**獨立計算**還是含在賠償內？
3. **故意行為**是否除外？（一定除外）
4. **告知義務違反**是否覆蓋？（多數含）
5. 跨院執業（兼職、支援）是否覆蓋？

### 保單常見「除外條款」（出事不賠）
- ❌ 故意行為、犯罪行為
- ❌ 酒醉、毒品狀態下執業
- ❌ 超出執業範圍（無照、跨科違法）
- ❌ 醫美廣告承諾結果（誇大宣傳）
- ❌ 病歷未誠實記載 / 偽造文書（自業）
- ❌ 未通知保險公司就私下和解

→ **接到糾紛 24 小時內通知保險公司**，是保單條款的義務。

## 四、出險流程（接到通知後）

```
[第 1 天]
  ├─ 通知個人責任險公司（出險通知電話）
  ├─ 院方法務同步通知
  └─ 不要私下回應家屬

[第 1 週]
  ├─ 保險公司指派律師（或你自己找律師後請保險覆蓋）
  ├─ 提交事件時間軸
  └─ 收集醫療文件（病歷、影像、鑑定）

[訴訟中]
  ├─ 律師費由保險覆蓋（依保單）
  ├─ 鑑定費由保險覆蓋
  └─ 重大協商必須通知保險公司

[判決]
  ├─ 賠償金保險公司理賠
  ├─ 上訴期間繼續覆蓋
  └─ 若超過保額，自負差額
```

## 五、PGY 階段的保險建議

### 必投
- **醫師公會團保**：通常每年 3,000-8,000 元
- 至少保額 500 萬

### 可選
- 個人加保高額責任險：1000-3000 萬
- 形象與媒體危機處理險

### 為什麼 PGY 也要保
- 你**會看病人、開醫囑、簽病歷** = 你是執業醫師 = 你會被告
- 急診、值班是 PGY 主戰場 = 高風險環境
- 訴訟可能拖 5-10 年 = **R5 / 主治階段才被告 PGY 期間的 case** 都很常見

## 七、常見 PGY 不知道的 4 件事

### 1. 你的「主治簽完字」也要小心
你寫了病歷後給主治醫師簽，主治醫師簽了 → **你和主治醫師都負責**。
但有時候主治醫師沒看仔細就簽，PGY 仍要承擔自己處置的責任。

→ **病歷寫得對 = 主治信你 + 法庭信你**。

### 2. 「會診紀錄」是你的盾牌
複雜病人會診其他科 → 病歷詳實寫**會診時間、會診醫師、會診意見**。
事後若被告：「我已會診 OO 科 OO 醫師於 OO 時間，建議 OO 處置，依其意見執行。」 = 責任部分轉移。

### 3. 「跨科處置」要先確認執業範圍
醫師法 §28 規定執業範圍。
PGY 受訓中 → 在「主治指導下」可進行跨科處置。
但**單獨進行**自己沒受訓的處置 = 風險高。

### 4. 「離職後病歷紀錄」仍要保存
你離開某醫院 → 該醫院仍要保存你寫的病歷 7 年（醫療法 §70）。
被告時要回去調病歷副本 → **離職前留下完整病歷影本**對自己最好（不違法，但要遵照院方流程）。

---

## 八、結論：兩道防線同時做好

1. **第一防線（防被告）**：病歷 + 告知 + SOP
2. **第二防線（防破產）**：個人責任險 + 醫師公會團保

這兩個都做好 = 你可以**安心當愛心醫生**而不怕被告毀掉人生。
