# 各科地雷 #1：急診（最高風險科別）

> 急診是醫療糾紛文獻一致認定的高風險前三名。
> 5 個最常被告的情境 + 真實案例對應 + 防身做法。

---

## 為什麼急診最容易被告

| 結構性原因 | 法律影響 |
|-----------|---------|
| 醫病關係短（30 分鐘決定生死）| 信任低 → 一有狀況直接告 |
| 多重診斷 / 不典型表現 | 任何漏診都是攻擊點 |
| 病情變化快 | 「漏看」很常見，鑑定容易抓 |
| 24 小時運作、值班疲勞 | 錯誤率 ↑ |
| 醫療資源有限 | 「設備未到位」也算違反義務 |

## 🚨 雷區 1：胸痛漏診 ACS

### 真實案例對應
- 沒有特定 case01-13 為胸痛案，但該情境是急診鑑定報告高頻內容

### 法官會問什麼
1. 「30 歲以上胸痛，有沒有 10 分鐘內 EKG？」
2. 「有沒有抽 TnI？0h + 3-6h 兩次？」
3. 「家族史、抽菸史問了嗎？」
4. 「正常 EKG 為什麼讓他離院？」

### PGY 防身做法
- **30 歲以上胸痛 → 10 分鐘內 EKG，不論長相多年輕**
- **EKG normal 不代表沒事**：糖尿病患、女性、老人常非典型
- TnI 抽兩次（0h + 3-6h），重複 EKG
- **HEART score** 評估（Heart-rate / EKG / Age / Risk / Troponin）
- 衛教單寫「紅旗症狀」：胸痛加劇、冒冷汗、呼吸困難 → 立刻回診

### 病歷救命寫法
```
S: 45 y/o 男性，胸前壓迫痛 30 min，無冒汗，否認家族史
O: BP 130/80, HR 78, EKG no ST change（時間: 10:15 attached）
   TnI 0h: <0.012, 3h pending
A: 胸痛 r/o ACS, HEART score 3（low risk）
P: 1. 觀察 + 重複 EKG + TnI 3h
   2. 給病人衛教單，告知紅旗症狀回診條件
   3. 如 TnI 3h 仍正常 + 症狀緩解 → 排心臟科 OPD f/u
```

## 🚨 雷區 2：頭部外傷漏診（CT 該不該做）

### 真實案例對應
- **case04 台大頭部外傷案**（病歷沒記 GCS → 重大瑕疵 → 舉證轉換）

### 法官會問什麼
1. 「嘔吐 2 次有沒有評估 CT？」
2. 「抗凝者有沒有做 CT？」
3. 「GCS 有沒有記錄？多久重複一次？」
4. 「離院前的衛教有沒有寫？」

### PGY 防身做法（用 Canadian CT Head Rule）

| 高風險特徵（任一即建議 CT）| 動作 |
|--------------------------|------|
| GCS < 15 (2 hr post-injury) | CT |
| 開放性骨折嫌疑 | CT |
| 頭蓋骨基底骨折跡象 | CT |
| 嘔吐 ≥ 2 次 | CT |
| **65 歲以上** | CT |
| **抗凝血劑使用** | CT（閾值更低）|
| 嚴重撞擊機制（車禍、墜樓 > 1 公尺）| CT |
| 30 分鐘以上意識不清 | CT |
| 順向性失憶 > 30 分鐘 | CT |

→ **不做 CT 也要寫理由**：「依 Canadian CT Head Rule 評估，無 CT 適應症，原因 OO」

### 病歷救命寫法
```
S: 28 y/o 男性，跌倒後 30 min，無意識喪失，不記得撞擊瞬間
O: GCS E4V5M6 = 15（10:00, 11:00, 12:00 三次 stable）
   無外傷、瞳孔等大反應正常、四肢力氣正常
A: 輕度頭部外傷，依 Canadian CT Head Rule 無 CT 適應症
P: 1. 觀察 4 小時
   2. 已給「頭部外傷出院衛教單」
   3. 已向家屬說明：24-48 hr 內若出現嗜睡、嘔吐、頭痛加劇、肢體無力 → 立刻回急診
   4. 家屬表示了解（家屬 王XX 簽名）
```

## 🚨 雷區 3：腹痛漏診（老人特別嚴重）

### 真實案例對應
- **case03 桃園醫院闌尾炎案**（後繼通知義務）

### 法官會問什麼
1. 「老人腹痛你做了什麼？只給止痛藥就讓他走？」
2. 「pain out of proportion 有沒有想 mesenteric ischemia？」
3. 「育齡女性有沒有測 β-hCG 排除 ectopic？」
4. 「CT 報告事後出來顯示有問題，你有通知病人嗎？」

### PGY 防身做法

**老人腹痛紅旗（任一即考慮 CT）**：
- > 65 歲
- 持續 > 6 小時
- 伴 vital sign 不穩
- 腹脹、無腸鳴音
- 既往心血管疾病

**育齡女性腹痛 → 必測 β-hCG**（避免漏 ectopic）

**「後繼通知義務」**（case03 確立的法律原則）：
- 病人離院後，影像報告才出來顯示異常 → **必須主動通知**
- 流程：打電話 → 家屬電話 → 留語音 → 全部記錄通話時間
- 即使聯繫不到，**「嘗試紀錄」**也是免責關鍵

### 病歷救命寫法
```
S: 75 y/o 男性，腹痛 8 hr，伴嘔吐 2 次
O: BP 145/85, HR 88, soft abdomen with mild RLQ tenderness
   無 rebound, 腸鳴音正常
   CT abdomen ordered（pending）
A: 老人腹痛 r/o appendicitis / mesenteric ischemia / AAA / sigmoid diverticulitis
P: 1. 等 CT 結果
   2. 若 CT 正常但症狀持續 → admit 觀察
   3. 若 CT 正常 + 症狀緩解 → 出院 + 24 hr OPD f/u（衛教 red flag）
   4. **若 CT 異常 + 病人已離院 → 立刻電話通知**
   5. 重要影像 review owner: 我（王 PGY，分機 1234）
```

## 🚨 雷區 4：嬰幼兒發燒漏診（最少人懂但風險高）

### 真實案例對應
- 兒童 sepsis / meningitis 漏診是急診的災難級風險

### 法官會問什麼
1. 「< 3 個月嬰兒發燒，有沒有依 SOP 完整評估？」
2. 「fever + irritability + bulging fontanelle 有沒有想 meningitis？」
3. 「LP 該做沒做的理由是什麼？」

### PGY 防身做法

**嬰兒發燒分齡管理**：
| 年齡 | 處置 |
|------|------|
| < 1 個月 fever | **必 admit + full sepsis workup（含 LP）** |
| 1-3 個月 fever | 多 admit；至少 CBC + UA + BC + 慎重評估 |
| 3-36 個月 fever | 看 toxic appearance，必要時做 sepsis workup |

**Toxic appearance signs**（任一即警戒）：
- Lethargy / poor feeding
- Bulging fontanelle
- Mottled skin
- Capillary refill > 2 sec
- Cold extremities

→ 父母說「我帶他去看門診就好」**不能直接放走** < 3 個月 fever 嬰兒。

## 🚨 雷區 5：拒絕治療病人（高風險 AAD）

### 法官會問什麼
1. 「病人 EKG 異常還讓他走，你有勸阻嗎？」
2. 「AAD 同意書有沒有寫具體風險？」
3. 「病人離院後死亡，你的勸告紀錄在哪？」

### PGY 防身做法

**「五不能讓他走」清單**（即使病人吵著走也要簽 AAD）：
1. 胸痛 + EKG/TnI 異常
2. 腹痛 + 急腹症徵兆（rebound, peritoneal sign）
3. GCS < 15 / 神經學異常
4. Vital sign 不穩（BP < 90, HR > 120, SpO2 < 92%, T > 38.5°C 老人）
5. < 3 個月嬰兒發燒

**AAD 三件套**：
1. **告知留院的醫療必要性**（具體風險，不是「會很嚴重」）
2. **簽 AAD 同意書**
3. **病歷詳實記錄**：說明了什麼、病人/家屬反應、決定離院原因

### 病歷救命寫法
```
05:30 病人陳XX 表示「我等等要上班，我要回家」
05:35 已向病人及配偶說明：
  - 您現在 EKG 顯示心臟病發作中
  - TnI 0.45 已確認心肌損傷
  - 不留院的具體風險：3 小時內可能猝死
  - 建議立即送 CCU + cath lab
05:40 病人仍堅持離院。已請夜班總值班醫師再次說明，病人理解但仍拒絕
05:45 病人簽自動出院書（AAD），配偶王XX 簽見證
05:48 已給病人衛教：若胸悶加劇、冒冷汗、呼吸困難 → 立刻回診或叫 119
```

## 急診醫師「下班前 5 個自我提問」

每結束一個 case 前問自己：

1. ✅ **病歷三段式**（看到/想/做）齊全嗎？
2. ✅ 紅旗症狀我有評估到嗎？
3. ✅ 該做的影像/抽血做了嗎？沒做的有寫理由嗎？
4. ✅ **離院衛教**有寫進病歷嗎？
5. ✅ pending 的影像/抽血誰負責 review？我有 follow up plan 嗎？

任一回答「沒有」 → 補上去再下班。
