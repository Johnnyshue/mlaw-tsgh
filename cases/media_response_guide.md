# 媒體應對指南：當記者在急診堵你時怎麼辦

> 醫療糾紛上社會新聞 → 記者堵在急診室、診間、家門口
> 你**亂講一句** = 全國頭條 = 訴訟敗訴關鍵
> 這份指南教你**保命的對話腳本**

---

## 為什麼這份指南很重要

### 媒體報導對訴訟的影響
- 病人/家屬看到報導 → 加深「我要告」的決心
- 法官 / 鑑定委員看到報導 → 心證受影響
- 你說的話被剪輯成「醫師承認過失」 → 證據

### 真實情境
台灣多數醫療糾紛大案（北榮、台大、馬偕等）都有上社會新聞。
醫師「在記者前不慎發言」 → 後續訴訟更難打。

## 黃金原則：3 不 + 5 要

### 🚫 3 不

#### 不 1：不要在「未經院方授權」下對記者說話
- 法律地位：**院方公關 / 主管才能對外發言**
- 你「個人意見」 = 個人責任
- 即使你說的是事實，剪輯後可能變另一個意思

#### 不 2：不要承認「過失 / 抱歉 / 我會負責」
- 醫療事故法 §7 保護「遺憾、道歉、讓步」**僅限於與當事人溝通的場合**
- 對媒體說 = **不在 §7 保護範圍**
- 「對不起讓您經歷這些」 = 對家屬說 OK；對記者說 = 變新聞標題「醫師道歉，承認疏失」

#### 不 3：不要用情緒回應
- 即使記者堵著你罵 → 不能回罵
- 即使被誤解 → 不要急著澄清
- 反應越激動 = 鏡頭越愛

### ✅ 5 要

#### 要 1：標準回應「請聯絡院方公關」
**對記者最安全的話**：
> 「關於這個事件，請依正常程序聯絡本院公關部，我**不便個別回應**。」

→ **背下來。記者堵你時就用這句。**

#### 要 2：要立刻通知院方
- 記者出現 = 立刻通知科主任、護理長、院方公關
- 院方有「**媒體統一發言窗口**」
- 你個人不應該成為發言人

#### 要 3：要保持移動
- 不要被堵在原地接受採訪
- 「我有病人要看，請先聯絡公關」+ 立刻離開
- 病人安全是你的義務 → 是合法理由

#### 要 4：要記錄記者言行
- 記者問了什麼、什麼時候、用什麼語氣
- 你回了什麼（盡量字面記憶）
- 事後寫成書面紀錄
- 萬一被剪輯抹黑 → 你有事實版本

#### 要 5：要尊重病人個資
- 即使記者套出「病人姓 XX，幾歲，做了 OO 手術」
- 你**一個字都不要說**
- 個資法 §6（特種個資）+ 醫師法 §23（保密義務）= 雙重防線

## 常見「陷阱問題」與標準回應

### Q1：「醫師你有什麼想對家屬說的？」
**❌ 錯誤回應**：「我感到很抱歉，我們會盡力處理...」（變標題：醫師道歉）
**✅ 正確回應**：「目前事件正在處理中，請依院方流程進行說明。」

### Q2：「請問當時為什麼這樣處置？」
**❌ 錯誤回應**：「因為當時情況比較複雜，我以為...」（給對方武器）
**✅ 正確回應**：「醫療詳情涉及病人個資，我不便對外說明，請聯絡院方公關。」

### Q3：「醫院有沒有疏失？」
**❌ 錯誤回應**：「我覺得我們已經盡力了...」（暗示有疏失）
**✅ 正確回應**：「醫療程序的評估有專業機制，請待專業鑑定結果。」

### Q4：「家屬說你態度很差，是這樣嗎？」
**❌ 錯誤回應**：「絕對沒有！我態度很好！」（被剪輯成爭吵畫面）
**✅ 正確回應**：「醫病溝通有院方紀錄，請聯絡院方公關獲得完整資訊。」

### Q5：「醫師可以給我看一下病歷嗎？」
**❌ 錯誤回應**：「等等，我打開給你看一下」（個資法 §41，5 年以下）
**✅ 正確回應**：「病歷涉及病人個資，依法不得對外揭露，請聯絡院方公關。」

### Q6：「你會繼續執業嗎？」（暗示你應該停業）
**❌ 錯誤回應**：「我會繼續看病人...」（變標題：醫師堅持執業，無視家屬）
**✅ 正確回應**：「個人執業與否，待事件處理結果後再做決定，目前以病人照護為優先。」

## 社群媒體：另一個雷區

### ❌ 千萬不要做的
- 在 IG / Facebook 限動發「今天好衰，被家屬告」
- 在 Line 群組討論個案（即使匿名）
- 在 Dcard / PTT 發文「這位家屬太誇張」
- 把病歷截圖（即使馬賽克）發出去

### 法律後果
- 個資法 §41：散布特種個資 → **5 年以下有期徒刑**
- 醫師法 §23：保密義務違反 → 行政處分 + 民事
- 妨害名譽（如果評論家屬）→ 民事
- **被肉搜後**訴訟更不利（家屬律師會把社群文當證據）

### 安全做法
- 想抒發情緒 → 找信任的人**面對面**講
- 想學習 case → 用「**完全去識別化**」（沒有任何可被認出的細節）
- 真要發文 → 先給律師朋友看過

```
記者：「您好，我是 OO 報的記者，請問是 OO 醫師嗎？關於今天 OO 病人的事件...」

✅ 你：「不好意思，我這邊是病房，不便接受採訪。
       媒體相關事務請聯絡本院公關部，
       電話 OO-OOOO-OOOO，謝謝。」

[掛斷]
```

→ **不要先聽記者問什麼再決定要不要回**。立刻轉介。


→ **不要刪除任何訊息**（即使覺得很煩）。**截圖存證**。

## 萬一被剪輯抹黑了怎麼辦？

### 第一時間
1. **不要立刻在社群反駁** — 反而擴大爭議
2. **保留原始素材** — 找出原始錄音、影像
3. **通知院方法律** — 準備發聲明
4. **諮詢律師** — 評估是否提毀謗訴訟

### 法律工具
- 妨害名譽（刑法 §310）
- 民事侵權（民法 §184）
- 個資法（如報導含可識別資訊）

→ **但訴訟對方是媒體**，影響範圍大，要謹慎評估。

## 結論：4 句保命口訣

1. 「**請聯絡院方公關**」 — 萬用
2. 「**涉及病人個資不便說明**」 — 拒絕病情問題
3. 「**待專業鑑定結果**」 — 拒絕「有沒有疏失」問題
4. 「**個人執業以病人照護為優先**」 — 拒絕情緒挑撥

把這 4 句背起來，記者怎麼問你都答得出來。

## 補充：對家屬的「正式說明」場合

不同於記者，對家屬的說明是**有義務的**（醫療事故法 §6）。
但要透過「院方關懷小組」進行，不要單獨進行。
詳見 `ch08_dispute.md` 與 `dialogues.js`「家屬情緒失控」情境。
