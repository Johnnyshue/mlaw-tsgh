# 各科地雷 #6：內科（不要以為內科就沒事）

> 內科訴訟數量雖不及外科婦產，但**告知義務違反 + 病歷不全 + 漏診**仍是高風險。
> 「我是內科不會被告」是 PGY 最危險的誤解。

---

## 內科訴訟的「沉默高風險」

| 因素 | 影響 |
|------|------|
| 慢性病多 | 長期處方錯誤累積 |
| 多重用藥 | 交互作用、副作用責任 |
| 病情進展慢 | 容易讓醫師鬆懈 |
| 「告知不充分」是內科最大雷 | 慢性病告知少做 |

---

## 💊 雷區 1：藥物過敏反應 / 藥物錯誤

### 法律邏輯
- 已知過敏給過敏藥 = **重大過失**（民事必賠 + 可能刑責）
- 未問過敏史 = 違反醫師法 §11（親自診察）+ 醫療法 §82（注意義務）

### PGY 防身做法

**開藥前必問**：
1. 過敏史（藥物 / 食物 / 環境）
2. 既往不良反應
3. 目前用藥（特別是抗凝血劑、降血壓藥、降血糖藥）
4. 腎功能 / 肝功能（劑量需調整）

**最常忽略的藥物交互作用**：
- Warfarin + 抗生素 / NSAIDs / 中草藥 → bleeding
- ACE inhibitor + spironolactone → hyperkalemia
- Statins + macrolides / fibrate → rhabdomyolysis
- SSRIs + tramadol → serotonin syndrome
- Benzodiazepines + opioids → respiratory depression（FDA black box）

### 病歷救命寫法
```
Pt: 70 y/o 男性，AF on warfarin INR 2.5, GFR 45
Today: URI symptoms, ?bacterial pharyngitis

Pre-prescription check:
- Allergy: NKDA confirmed
- Renal function: GFR 45 → adjust antibiotic dose
- Drug interaction: warfarin + 多種抗生素 → bleeding risk
  → 選擇 amoxicillin（與 warfarin 交互作用相對較低）
  → **告知病人：增加抗生素期間需密切監測 INR，1 週內回診抽 INR**

Rx: amoxicillin 500mg PO TID × 7d
Plan: INR check 1 week, return if any bleeding
```

## 💊 雷區 2：心肌梗塞 / 主動脈剝離漏診

### 真實案例對應
- 內科門診 / 病房漏診 ACS / dissection 是經典訴訟

### PGY 防身做法

**胸痛紅旗（門診版）**：
- 突發劇烈胸痛輻射到背
- 撕裂痛感（dissection 典型）
- 男性 > 50 歲、女性 > 60 歲
- 糖尿病 / 高血壓 / 抽菸
- 家族猝死史

**門診胸痛 → 不要試圖「先吃藥觀察」**
- 立刻 EKG（門診若有）
- 不確定 → **直接送急診**
- 病歷記錄：「已告知病人胸痛性質可能為急性冠心病，建議立即至急診評估，病人 OO 並前往」

**「我以為是肌肉酸痛」是內科醫師最常後悔的話**

## 💊 雷區 3：糖尿病併發症告知不足

### 法律邏輯
- 糖尿病管理涉及長期決策
- 「沒告訴我會這樣」是糖尿病糾紛常見訴詞

### PGY 防身做法

**新診斷糖尿病要告知的事項**：
1. **疾病性質**：慢性、進展性、需終身管理
2. **併發症風險**：
   - Microvascular：retinopathy（失明）、nephropathy（洗腎）、neuropathy（截肢）
   - Macrovascular：心血管事件（MI、stroke）
3. **治療目標**：HbA1c < 7%（個別化）
4. **生活型態介入**：飲食、運動、戒菸
5. **追蹤計畫**：定期 HbA1c、眼底檢查、腎功能、足部評估

**用藥告知**：
- Metformin：lactic acidosis 風險（GFR < 30 禁用）
- SGLT2i：euglycemic DKA、生殖道感染、截肢風險（FDA warning）
- GLP-1 RA：胃腸不適、罕見胰臟炎

### 病歷救命寫法
```
新診斷 T2DM 病人 OPD 衛教

S: 55 y/o 女性，HbA1c 8.5%, FBS 180
A: T2DM (newly diagnosed)

** Education provided **
1. Disease overview: chronic, progressive
2. Complications: 已說明 retinopathy / nephropathy / neuropathy / CVD risks
3. Lifestyle: 減重 5-10%, 每週運動 150 min, 飲食衛教單已給
4. Pharmacology: metformin + lifestyle, target HbA1c < 7%
5. Follow-up: HbA1c q3M, BUN/Cr q6M, eye exam yearly, foot exam yearly
6. Red flags: hyper/hypoglycemia symptoms, foot lesions
7. Patient and spouse acknowledge understanding

Plan:
- Metformin 500mg PO BID
- 1 month follow-up (drug tolerance, lifestyle modification)
```

## 💊 雷區 4：抗凝治療出血併發症

### 法律邏輯
- Warfarin / NOAC 引起出血（特別 ICH）= 高賠償案件
- 「告知不充分 + 未及時調整」是常見敗訴原因

### PGY 防身做法

**抗凝告知 checklist**：
1. **適應症說明**（為什麼要抗凝）：AF、DVT、機械瓣膜
2. **出血風險**：minor bleeding 5-10%/year, major bleeding 1-3%/year, ICH 0.5%
3. **藥物交互作用**：
   - Warfarin: 食物（vitamin K）、抗生素、NSAIDs
   - NOAC: P-gp inhibitor、CYP3A4 inhibitor
4. **監測**：
   - Warfarin: INR q4W initially, q3M stable
   - NOAC: 腎功能 q6M
5. **緊急情況**：跌倒撞頭、明顯出血 → 立刻急診

**病歷寫法**：
```
已向病人及家屬詳細說明:
1. 抗凝治療目的（AF, CHA2DS2-VASc 4 → annual stroke risk 4%）
2. Major bleeding risk 約 1-3%/year, ICH 0.5%
3. 跌倒撞頭 → 立刻急診（不論有無症狀）
4. 牙科 / 手術前 7 天告知，由我評估是否需 bridging
5. 密集監測必要性
```

## 💊 雷區 5：感染症診斷遲緩 / 抗生素濫用

### 法律邏輯
- 敗血症延誤治療 → 高死亡率 → 訴訟風險
- **濫用抗生素**也是法律問題（健保申報違規 + 醫師法 §28）

### PGY 防身做法

**Sepsis 診斷標準**（記得 qSOFA）：
- RR ≥ 22
- Altered mental status (GCS < 15)
- SBP ≤ 100 mmHg
- 任 2 項 = 高死亡風險

**Sepsis bundle 1 hr**：
- Lactate level
- Blood cultures × 2 sets（before antibiotics）
- **Broad-spectrum antibiotics**（根據可疑來源）
- IV crystalloid 30 mL/kg if hypotension or lactate ≥ 4

→ **延誤每 1 hr，死亡率上升 4-9%**

**抗生素開立**：
- 適應症明確（不要為「咳嗽」開）
- 培養先抽再給藥
- De-escalation 後續調整
- 健保申報依規定

## 💊 雷區 6：腎功能 / 肝功能調整劑量遺漏

### 法律邏輯
- GFR / 肝功能不佳病人沒調整劑量 → 毒性 → 訴訟
- 教科書清楚記載「需調整」的，沒調整 = 違反醫療常規

### PGY 防身做法

**腎功能調整必查藥物**：
- 抗生素：vancomycin、aminoglycoside、cephalosporin、fluconazole
- 化療藥：cisplatin、methotrexate
- 心血管：digoxin、enoxaparin、fondaparinux
- 神經：gabapentin、pregabalin、lithium
- DM：metformin (GFR < 30 禁)、 SGLT2i

**查詢工具**：
- UpToDate Lexicomp
- Sanford Guide
- 藥師諮詢（**最強盾牌**）

→ 開高風險藥前**諮詢藥師**並記錄在病歷

## 內科醫師「門診/查房 5 個自我提問」

1. ✅ 過敏史 + 用藥史**親自確認**了嗎？
2. ✅ 紅旗症狀（胸痛、突發頭痛、腹痛）**有沒有評估**？
3. ✅ 慢性病告知（併發症、生活型態、紅旗症狀）寫了嗎？
4. ✅ 腎/肝功能對應**藥物劑量調整**了嗎？
5. ✅ 追蹤計畫（下次什麼時候回診、什麼情況立刻回診）**寫進病歷**了嗎？

## 病例：65 歲糖尿病慢性病管理（病歷救命寫法）

```
S: 65 y/o 女性, T2DM 10 yrs, HTN, hyperlipidemia
   Meds: metformin 1000mg BID, lisinopril 10mg, atorvastatin 20mg

O: BP 138/82, BMI 26
   HbA1c 7.2%, GFR 65, LDL 95
   Eye exam (last year): mild non-proliferative retinopathy
   Foot exam: intact monofilament, no ulcers

A: T2DM, fair control with mild retinopathy
   HTN, well controlled
   Hyperlipidemia, at goal

P: 1. Continue current medications
   2. Reinforce lifestyle: diet log review next visit
   3. Eye exam yearly (next: 2026/12)
   4. Foot exam every visit
   5. HbA1c q3M, lipid q6M
   6. **Patient education today**:
      - Goal HbA1c < 7%, achieved
      - Hypoglycemia symptoms reviewed
      - When to call: hypoglycemia, foot lesion, vision change
   7. Follow up 3 months
```

→ 這份病歷的優勢：每個面向都有，**法庭上等於把所有義務都做到了**。
