# 各科地雷 #2：外科（含骨科、神外、心外）

> 外科是訴訟最高險科別之一。骨科尤其在民事案件中常年第一名（佔 10%）。
> 5 個最常被告的情境 + 真實案例對應 + 防身做法。

---

## 🩹 雷區 1：手術同意書「不夠個別化」

### 真實案例對應
- **case02 新竹馬偕**（術前未做闌尾炎鑑別診斷 + 同意書未個別化）
- **case08 葳亞娜**（隆鼻惡性高熱，同意書未問家族史）

### 楊秀儀大師的核心觀點
「簽了不等於告知，**法律問你有沒有講，不是有沒有簽**」（master_yang_xiuyi.md 觀點一）

### PGY 防身做法

**手術同意書三段式**：
1. **印刷版內容**（醫療法 §63 法定項目）：手術名稱、原因、成功率、可能併發症
2. **個別化補充**（最容易漏）：
   - 病人特殊風險（年紀、基礎疾病、過敏、用藥）
   - 該病人不選 A 改選 B 的理由
   - 「替代方案」與「不治療的後果」
3. **病歷紀錄補強**（同意書沒寫的，病歷補）：
   - 已用模型/圖示說明
   - 家屬反應、提問內容
   - 給予思考時間（**不是進手術房才簽**）

### 病歷救命寫法
```
2026-04-30 OPD（術前說明）
S: 65 y/o 女性，膽結石反覆膽絞痛，已接受 3 次保守治療無效
O: 同前次門診，腹超：multiple gallstones with mild GB wall thickening
A: Symptomatic cholelithiasis, 建議 lap chole

** 已向病人 + 配偶（陳XX）詳細說明 **
1. 手術原因：反覆膽絞痛影響生活
2. 手術方式：腹腔鏡膽囊切除，4 個 trocar
3. 成功率：>95%（單純膽結石）
4. 主要風險：
   - 出血需轉開腹（< 1%）
   - 膽管損傷（0.3-0.5%）
   - 手術部位感染（2-3%）
   - 麻醉相關（< 0.1%）
   - 病人 65 歲 + 高血壓，加註 cardiac event risk
5. 替代方案：保守治療（症狀持續惡化）/ 不治療（膽囊炎、膽管炎風險）
6. 病人提問「能不能不開刀」 → 已說明保守治療失敗風險
7. 已給予 24 小時思考，明日回診再決定

P: 1. 病人/配偶表示理解
   2. 約 5/2 入院，5/3 開刀
   3. 同意書病人/配偶共同簽名
```

## 🩹 雷區 2：術後照護紀錄不足

### 真實案例對應
- **case10 黃慧夫案**（亞東術後筋膜切開術，末梢血液循環記載不足 → 二審有罪）

### 楊坤仁大師的核心觀點
「病歷是寫給未來法官看的，不是給自己看的」（master_yang_kun_ren.md 觀點二）

### PGY 防身做法

**術後 checklist（每天記錄）**：
| 項目 | 怎麼寫 |
|------|--------|
| Vital sign | BP/HR/RR/SpO2/T 具體數字 |
| 傷口狀態 | clean / mild redness / no discharge / sutures intact |
| **末梢血液循環** | 脈搏（dorsalis pedis +2/+2）、色澤（pink）、溫度（warm）、感覺（intact） |
| 引流量 | 具體 mL（10 mL serous） |
| 疼痛分數 | NRS 0-10 |
| 飲食/排泄 | tolerate clear liquid / passed flatus |
| 異常事件 | 任何發燒、出血、傷口問題 → **時間戳記 + 處置** |

### 為什麼末梢血液循環特別重要
case10 黃慧夫案核心爭議：截肢病人事後質疑「術後沒監測末梢循環」。
- 二審判 4 個月（有罪）：因為術後紀錄不足
- 再審無罪：靠「轉院後膝關節脫臼 = 因果關係斷鏈」翻案，**運氣很好**

→ 不要靠運氣翻案。**寫好病歷**遠比再審容易。

## 🩹 雷區 3：術中異物殘留

### 真實案例對應
- 紗布殘留、器械斷裂、計數錯誤 — 這類案例在文獻中常見，多以民事敗訴收場

### 法律邏輯
- **客觀違反醫療常規**（標準術後計數應為 0 caputure）
- 通常**直接適用民事過失**（不需要鑑定就能認定）
- 即使按 §82 是醫師個人，但醫院（§82 V）一定要賠

### PGY 防身做法
- **計數三次**：開腹前、關腹前、皮膚閉合前
- 計數不對 → **暫停，找 X 光**
- 異常物品紀錄在開刀紀錄
- **「count correct」是手術紀錄必寫項目**

## 🩹 雷區 4：開錯部位 / wrong-side surgery

### 真實案例對應
- 「左/右」混淆是骨科、神外、眼科共同雷區
- 多數醫院已規定 surgical site marking 為 SOP

### PGY 防身做法

**Universal Protocol（WHO 提倡）**：
1. **Pre-procedure verification**（術前驗證）：病歷、影像、X 光、知情同意書一致
2. **Site marking**（手術部位標記）：用不褪色筆，在意識清楚時讓病人本人指認
3. **Time-out**（手術前暫停）：手術房內所有人停下，再次確認

→ 病歷寫「已執行 universal protocol，site marked, time-out performed」

## 🩹 雷區 5：術後病情惡化未及時處理

### 真實案例對應
- **case12 肝硬化未送 ICU 案**（雖然無罪，但路徑教訓）
- **case13 抽脂後讓病人離院**（刑事入獄 2 年）

### 黃清濱大師的核心觀點
「醫療失敗 ≠ 醫療過失」，但**沒監測 = 你沒辦法主張你做了標準該做的事**

### PGY 防身做法

**術後監測標準（依手術等級）**：
| 手術等級 | 監測 |
|---------|------|
| Major surgery（剖腹、心胸）| ICU 至少 24 hr 或 PACU + 加護病房 |
| Moderate（lap chole、闌尾切除）| PACU + 一般病房嚴密監測 4 hr |
| Minor（腋窩淋巴結切除）| PACU 30 min - 2 hr |
| **Day surgery / 醫美**| **必須符合 discharge criteria（Aldrete score）才能離院** |

**Discharge criteria（Aldrete score ≥ 9）**：
- 活動：能依指令動 4 肢
- 呼吸：能深呼吸咳嗽
- 循環：BP 與術前 ± 20%
- 意識：清醒
- 氧合：room air SpO2 ≥ 92%

→ case13 抽脂案就是讓 Aldrete < 9 的病人離開診所，導致 3 小時後死亡。**入獄 2 年**。

## 各次專科特殊雷區

### 骨科
- **延誤診斷骨折**（X 光看漏 fracture line）
- **骨折手術 nerve injury**（告知義務 + 術中監測）
- 老人髖部骨折 → 24-48 hr 內手術（延誤可能加重併發症）

### 神外
- **頭部外傷 ICH 漏診**（case04 教訓）
- **脊椎手術 nerve injury**：必有完整告知 + 術中神經監測
- DBS、cranio 等高風險手術，同意書要超個別化

### 心外
- **CABG 術前評估完整性**（cardiac function、coronary anatomy）
- **CPB 相關併發症告知**（stroke risk 1-3%）
- **危急情況 emergency CABG** 與 emergency consent 程序

### 整形/醫美（民事訴訟風險最高）
- **成功率不能保證 100%**：「療效不如預期」是常見訴訟原因
- **術前術後照差異**：客觀記錄
- **侵入性麻醉必須有麻醉科醫師**（case08 葳亞娜教訓）
- **不得讓病人術後在無監護下離開**（case13 教訓）

## 外科醫師「術前 5 個自我提問」

進手術室前自問：

1. ✅ 同意書有**個別化**嗎？病人特殊風險寫了嗎？
2. ✅ 有沒有給病人**充分思考時間**？
3. ✅ Site marking 做了嗎？time-out 走了嗎？
4. ✅ 術後監測計畫**寫進醫囑**了嗎？
5. ✅ 急救/應變預案備齊了嗎？（出血、過敏、心律失常）
