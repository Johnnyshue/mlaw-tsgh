# 📄 病歷紀錄範本庫（直接複製貼進 HIS）

> 9 種最常用的高風險紀錄範本。
> 把這些**動詞 + 句型**內化成你的病歷習慣，95% 法律保護就到位。

---

## 範本 1：📞 會診紀錄（最強盾牌）

```
[YYYY-MM-DD HH:MM] 會診紀錄

主動會診原因：
本病人因 OO 症狀 + XX 病史，超出本科處置範圍 / 需要 OO 專科意見。

會診對象：OO 科 OO 醫師（分機 XXXX）

會診時間：HH:MM 撥電話 / line / 親自會面

OO 醫師評估意見（會診者口述）：
1. 評估診斷：OO
2. 建議處置：OO
3. 後續追蹤：OO

我的執行：
依會診意見執行 OO 處置，已將會診醫師意見記錄於本病歷。

PGY 王 XX 簽 / HH:MM
```

→ **法律意義**：會診紀錄完整 = 部分責任轉移到會診醫師。

## 範本 2：🚪 自動出院（AAD）紀錄

```
[YYYY-MM-DD HH:MM] AAD 紀錄

病人狀況：
- 意識：清醒、定向感正常、能回答問題
- 病情：OO 診斷，目前 vital sign 為 BP/HR/RR/SpO2
- 我的評估：仍需住院觀察 / 治療

已說明（口頭 + 書面）：
1. 留院的醫療必要性：OO
2. 不留院的具體風險：OO 小時內可能發生 XX、YY、ZZ
3. 替代方案：可選擇 OO（門診追蹤）、XX（轉院）

病人/家屬反應：
- 病人表示：「OO」
- 家屬（OO，與病人關係 OO）表示：「OO」
- 已給予思考時間：X 分鐘

決定：
病人在意識清楚下，自願選擇出院。
已簽署「自動出院同意書」。

衛教：
若出現 OO、XX、YY 紅旗症狀，立即回急診或撥 119。
已給予衛教單張。
明日 OO 點 OO 科 OO 醫師門診回診時間。

PGY 王 XX 簽 / HH:MM
護理師見證：OO 簽
家屬見證：OO 簽
```

## 範本 3：🪦 DNR / 不予急救紀錄

```
[YYYY-MM-DD HH:MM] DNR 簽署紀錄

病人狀況評估：
- 病人 OO 歲，診斷 OO（末期/不可逆昏迷/植物人/極重度失智 — 依病人自主權利法 §14）
- 預期生存：OO（依文獻 / guideline）
- 治療反應：OO

病人意願（如能表達）：
病人在意識清楚時表示：「OO」
（紀錄日期、時間、見證人）

家屬討論：
參與者：OO（配偶）、XX（子女 N 人）、YY（兄弟姐妹）
討論內容：
- 已說明病情、治療選項、預後
- 已說明積極治療 vs 安寧緩和的差異
- 家屬達成共識：選擇 OO

簽署：
依安寧緩和醫療條例 §7：
□ 病人本人簽（如能）
□ 法定順位家屬簽（無病人意願時）

醫師簽署人：OO 主治醫師（依條例需要兩位醫師確認）
PGY 協助：王 XX 簽
```

→ 注意：DNR 與「撤除維生醫療」適用不同程序（§7 vs §7-1）。

## 範本 4：📝 知情同意（手術 / 侵入性檢查）

```
[YYYY-MM-DD HH:MM] 術前說明紀錄

病人：OO，OO 歲，特殊狀況：HTN/DM/抗凝中（個別化）

已詳細說明：
1. 手術原因：OO
2. 手術方式：OO（具體步驟）
3. 預期成功率：OO%
4. 主要風險（依發生率排序）：
   a. 出血（< 1%）
   b. 感染（2-3%）
   c. 麻醉相關（< 0.1%）
   d. 個別化風險（依本病人狀況）：OO（OO%）
5. 替代方案：
   a. 保守治療：OO
   b. 不治療的後果：OO
   c. 其他術式：OO
6. 術後預期：恢復時間、需要的後續治療

病人/家屬反應：
- 病人表示理解，提出 OO 問題，已回答
- 家屬（OO）表示理解，同意手術

簽具：
- 手術同意書 / 麻醉同意書 / 侵入性檢查同意書
- 簽具人：病人 / OO（法定順位 OO）
- 給予思考時間：X 小時 / 至明日

PGY 王 XX 親自說明 / 主治醫師 OO 醫師確認
```

## 範本 5：🚨 急症處置紀錄（黃金 1 小時）

```
HH:MM:SS 病人 OO 出現 OO 症狀

HH:MM:SS Vital sign：BP/HR/RR/SpO2/T
HH:MM:SS 緊急評估：ABC stable / not stable
HH:MM:SS 啟動 OO bundle / ACLS / sepsis bundle

HH:MM:SS 處置 1：OO（劑量、途徑）
HH:MM:SS 處置 2：OO

HH:MM:SS 通知主治：[已 line / 已電話 / 已親自報告] OO 醫師
                  → 主治回應：「OO」（HH:MM 接到指示）
                  → 如未聯絡到：HH:MM 通知總值班 OO 醫師

HH:MM:SS Vital sign re-check：[數字]
HH:MM:SS 處置 3：OO

HH:MM:SS 病人狀況改善 / 持續惡化 → 升級至 ICU

簽：PGY 王 XX
```

→ **重點**：每個動作都有時間戳記，鑑定看到時序完整 = 你做得對。

## 範本 6：💊 藥物錯誤事件報告

```
[YYYY-MM-DD HH:MM] 藥物事件報告

事件描述：
HH:MM 發現 OO 病人 [漏給 / 多給 / 錯藥 / 過敏]：原處方 OO，實際 XX

立即處置：
1. HH:MM 立刻 OO（停藥 / 給 antidote / 監測）
2. HH:MM 通知主治：OO 醫師 → 評估後續監測 OO 小時
3. HH:MM 通知病房護理長 OO，啟動 incident report

對病人的告知：
HH:MM 已向病人 + 家屬說明：
- 發生了什麼
- 我們已採取 OO 處置
- 接下來會 OO 監測
- 病人/家屬反應：OO

事件後續：
- 院方藥物事件處理小組已通報
- 個人責任險已通知（如重大事故）
- 病歷詳實記錄事件經過

PGY 王 XX 簽
```

→ **主動通報的醫師很少被告，掩蓋的醫師都被告**。

## 範本 8：📵 主治不在場時的緊急處置

```
[YYYY-MM-DD HH:MM] 主治聯繫不上之緊急處置紀錄

事件起點：
HH:MM 病人 OO 出現 OO 急症（vital sign / GCS / 主訴）

聯繫主治：
HH:MM 第一次撥打主治醫師 OO 手機 → 未接通
HH:MM line 訊息已送出 → 未讀
HH:MM 第二次撥打 → 進語音信箱
HH:MM 撥打主治家用電話 → 未接通

升級流程：
HH:MM 通知總值班醫師 OO（電話 OO）
       → 總值班指示：OO
HH:MM 同時依 sepsis bundle / ACLS / OO SOP 即時處置：
       1. OO（劑量、時間）
       2. OO
HH:MM 主治於 HH:MM 來電，已說明病人狀況，主治指示 OO

處置依據：
依當時臨床判斷 + OO guideline + 醫療法 §82 第 4 項（合理臨床裁量考慮工作條件）

PGY 王 XX 簽
```

→ **法律保護**：你已盡最大努力聯繫，且依 SOP 處置，醫師法 §21 義務已盡。

## 範本 9：📋 出院前衛教紀錄

```
[YYYY-MM-DD HH:MM] 出院衛教紀錄

出院時病人狀況：
- Vital sign 穩定（具體數字）
- 主要診斷：OO
- 出院時用藥：OO

已衛教（口頭 + 書面）：
1. 用藥方式、劑量、時間
2. 飲食 / 活動限制
3. 紅旗症狀回診條件：
   - OO（如：胸痛加劇、呼吸困難）→ **立即回急診**
   - XX（如：發燒 > 38.5°C）→ **24 小時內回診**
   - YY（如：傷口紅腫）→ 門診回診
4. 預定回診：OO 醫師門診（YYYY-MM-DD HH:MM）
5. 急救聯絡：本院急診 OO / 119

家屬同意：
病人 + 家屬（OO）表示理解所有衛教內容。
已給予衛教單張一份。
家屬簽名：OO

PGY 王 XX 簽
```

→ **法律意義**：醫療法 §81 告知義務 + 後繼義務的書面證據。

## 通用頭尾範本（每個紀錄都加）

### 頭部（Top）
```
[YYYY-MM-DD HH:MM:SS] 紀錄類型
病人：OO（病歷號 OOOO）
紀錄者：PGY 王 XX
```

### 尾部（Bottom）
```
本紀錄為親自記載，符合醫療法 §68 規定。
若有修改，依 §68 II 以畫線保留原文 + 註明修改時間。

PGY 王 XX 親簽 / HH:MM
（如必要）主治確認：OO 醫師 簽
```

## 「3 個動詞」永遠記得寫

任何病歷都要有：

1. **「已告知 / 已說明」** — 證明你盡告知義務
2. **「已記錄 / 已紀錄」** — 證明你做了書面義務
3. **「已通知 / 已聯繫」** — 證明你升級求援

→ **三動詞齊全 = 你的病歷自己會幫你說話**。
