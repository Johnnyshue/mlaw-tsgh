# 防身指南 08｜病人/家屬常見要求 SOP（每個值班都會遇到）

## 來源
- 醫療法 §63、§71、§74（同意/病歷複本）
- 個資法 §3 §15（當事人權）
- 病主法（預立醫療決定）
- 安寧緩和醫療條例（DNR）
- 信心等級：高

---

## 一、白話結論

**「答應前先想：法律授權給誰？」**
病人本人的權利，不能任由家屬代行（除非本人意識不清）。

## 二、10 大常見要求 SOP

### 1️⃣ 「我要看病歷／我要影印病歷」
- ✅ 法源：醫療法 §71、§74
- ✅ 病人本人 → **必須提供**（最遲 30 日內）
- ⚠️ 家屬要求 → **病人不在場 + 沒授權書 = 拒絕**
- ⚠️ 病人死亡後 → 配偶/直系血親可申請（法院實務見解）
- 收費：影本費合理（約 5 元/張）+ 行政費

**SOP**：
```
1. 確認是病人本人 → 出示證件
2. 不是本人 → 要授權書 + 病人簽名 + 證件影本
3. 病人死亡 → 配偶/父母/子女 + 戶籍謄本
4. 拒絕特殊資訊？病主紀錄、HIV、精神科 → 仍須提供（法律無禁止）
```

### 2️⃣ 「我要錄音／我要錄影」
- 病人**對「自己參與的對話」錄音 → 合法**（不違反通保法）
- 病人錄「不是自己參與的對話」 → 違法（刑法 §315-1）
- 病人**錄你 → 你不能拒絕**，但可請他「告知一聲」表示尊重
- ⚠️ **你不要錄病人**（除非醫病溝通、明確告知 + 同意）

**SOP**：
```
1. 病人說「我要錄」 → 「我尊重，我們會繼續說明」
2. 不要中斷對話表現「我有什麼好藏」
3. 把對話內容做成書面 → 病歷對照
4. 病人錄完後若剪輯造謠 → 反告誹謗（刑法 §310）
```

### 3️⃣ 「我要看監視器」
- 醫院監視器屬醫院個資 → **病人/家屬非當然有權調閱**
- 法定途徑：警察、檢察官、法院 → 醫院必須提供
- 院方做法：請司法機關開公文

**SOP**：「監視器涉及他人個資，需司法機關依法調閱。」

### 4️⃣ 「我要 DNR／不要急救」
- ✅ 法源：安寧緩和醫療條例 §7
- 條件：
  - 病人本人意識清楚 → 簽署「不施行心肺復甦術同意書」+ 兩位專科醫師判定為末期
  - 病人意識不清 → 最近親屬簽（法定順位：配偶 → 成年直系 → 父母 → 兄弟姐妹 → 祖父母）
- ⚠️ 家屬意見不一 → **以法定順位前者為準**
- ⚠️ 病人若有預立 AD → 優先於家屬意見（病主法）

### 5️⃣ 「我要 AAD／我要回家」
- 病人有自主權（醫療法 §63 反向解釋）
- 但你不能「就讓他走」
- **SOP**：
  ```
  1. 評估意識能力（精神狀態、認知）
  2. 告知不治療的具體後果（紅旗、死亡風險）
  3. 提供替代方案（門診追蹤、其他醫院）
  4. 簽 AAD 同意書（自願出院）
  5. 病歷寫詳細：勸告內容、病人回應
  ```
- ⚠️ 病人意識不清 / 自殺意念 → **不能讓他走**（精神衛生法 §41 強制）

### 6️⃣ 「我要轉院」
- 醫療法 §73：「醫院、診所因限於人員、設備及專長能力，無法確定病人之病因或提供完整治療時，應建議病人轉診。」
- 條件達成 → 你**有義務**協助轉院
- 危急病人 → §73 II「應先予適當之急救處置」
- 家屬要求轉院 → 你判斷不適合 → 病歷寫風險告知 + 簽 AAD

### 7️⃣ 「給我開診斷書／證明書」
- 醫師法 §17：醫師「**親自診察**」才能開
- ⚠️ 沒看過病人 → 不能開（密醫罪）
- ⚠️ 改死因 / 改診斷 → 偽造文書（刑法 §215，3 年以下）
- ⚠️ 「請假診斷書」誇大 → 仍違 §17、§215
- 收費：依各縣市衛生局公告

### 8️⃣ 「我要病人轉到 ICU／加做 OOO」
- 醫療指令權在醫師，不在家屬
- 家屬要求 → 你判斷不必要 → 解釋 + 病歷紀錄
- 家屬要求 → 你判斷有必要但院方資源不足 → 通報院方 + 紀錄
- ⚠️ 家屬施壓「不送 ICU 出事你負責」 → 不必妥協，**鑑定看的是醫療常規不是家屬要求**（參 Case 12）

### 9️⃣ 「我要某某藥／某某檢查」
- 健保給付限制 → 不在 indication 內 → 拒絕（自費除外）
- 自費 → 病人有要求權 + 你有判斷權
- ⚠️ 為討好病人開不必要檢查 → 健保詐欺（健保法 §81，5 倍罰鍰 + 停約）

### 🔟 「醫師我要私下找你（LINE/電話）」
- ⛔ **不要單獨給私人聯絡方式**
- 醫病關係的對話應在「正式管道」
- 例外：你願意提供「主治信箱/院內分機」可以
- ⚠️ 給 LINE → 半夜訊息 → 沒回 → 病人出事 → **被告「告知義務違反」**

## 三、口袋速查（一句話版）

| 要求 | 一句話 |
|---|---|
| 看病歷 | 「病人本人請出示證件，不是本人請帶授權書」 |
| 錄音 | 「您可以錄，我們繼續說明」 |
| 看監視器 | 「需司法機關依法調閱」 |
| DNR | 「需兩位專科醫師判定末期 + 法定順位簽署」 |
| AAD | 「我會告知風險、簽同意書，病歷會詳實紀錄」 |
| 轉院 | 「依醫療法 §73 我會協助安排」 |
| 開假證明 | 「我必須親自診察，我不能開」 |
| 加做檢查 | 「我評估後沒有適應症，您可考慮自費」 |
| 給 LINE | 「請走院內回診/分機聯繫」 |
