# 法律分析 08｜再生醫療雙法（2024/06/04 三讀、2024/06/19 公布）

## 來源
- 再生醫療法

- 衛福部 2024/06/04 三讀通過、2024/06/19 公布
- 衛福部官方說明：https://www.mohw.gov.tw/cp-16-78933-1.html
- 信心等級：中–高（細部子法仍在預告）

---

## 一、白話結論

**「醫師私下幫病人打幹細胞 / 細胞療法」 → 違法。**
2024 年新法之後，**再生醫療必須在主管機關核准的醫療機構執行**，違者最高罰 **2,000 萬**。

---

## 二、為什麼有「雙法」

| 法案 | 規範範圍 |
|---|---|
| **再生醫療法** | **技術面** — 醫療機構執行再生醫療的資格、醫師資格、細胞來源、廣告 |
| **再生醫療製劑條例** | **產品面** — 細胞製劑的查驗登記、上市後監視、附款許可 |

→ 一個管「醫療行為」、一個管「藥品（製劑）」。

---

## 三、什麼算「再生醫療」？

依再生醫療法 §3，包含：
1. **細胞治療**（CAR-T、間質幹細胞、自體免疫細胞 DC、CIK、NK 等）
2. **基因治療**（基因編輯、基因傳遞）
3. **組織工程**（軟骨、皮膚、角膜重建）
4. 其他經中央主管機關公告之再生醫療技術

---

## 四、醫師最關鍵的 5 個合規重點

### 1️⃣ 必須在「核准的醫療機構」執行
- 再生醫療法 §10：醫療機構必須經中央主管機關 **查核及許可** 才能執行
- 沒許可就執行 → §32 罰 100 萬–2,000 萬
- 個人診所執行細胞療法 → 高風險違法

### 2️⃣ 醫師資格條件
- §11：執行再生醫療的醫師必須具備：
  - 相關專科醫師
  - 完成中央主管機關指定之教育訓練
- 不符資格 → 罰鍰 + 停業

### 3️⃣ 細胞來源管理
- 自體 / 異體 → 都要 **知情同意 + 倫委會審查**
- 細胞提供者招募廣告需事前審查
- 違反 → 罰鍰 + 細胞銷毀

### 4️⃣ 「附款許可」制度（製劑條例特色）
- 危及生命 / 嚴重失能疾病 → 可在 **完整三期試驗前**附款許可上市
- 條件：證明安全性 + 初步療效
- 上市後仍必須持續做試驗 + 安全監視
- 例：CAR-T 治療某些罕病、ALS、重度脊髓損傷

### 5️⃣ 廣告與招募
- 細胞療法廣告 → **嚴格審查**
- 不可宣稱「治癒」「神效」
- IG/FB 個人「分享案例」 → 視同廣告（§39）
- 違規最高罰 200 萬

## 五、住院醫師 PGY 的影響

### 醫學中心 PGY
- 看到 CAR-T / DC 細胞病人 → 都會在「核准的醫療機構」內 → 合法
- 注意 **知情同意書要完整**（細胞來源、風險、替代治療）
- 病人併發症（cytokine release syndrome、neurotoxicity）→ 通報 ADR

### 在區域 / 地區醫院的 PGY
- 看到自費「幹細胞療法」廣告 → **先查醫院是否核准**
- 沒核准的「美容診所幹細胞」 → 違法 → 不要參與
- 主治約你 moonlight 打細胞療法 → **先確認場所核准 + 你資格**

### 在診所 / 醫美打細胞療法
- ⚠️ 大多數診所**沒有核准** → 高風險違法
- 違反 §32 → 罰 100–2,000 萬
- 醫師個人 + 機構皆罰

## 六、與既有法規的銜接

| 既有法 | 銜接 |
|---|---|
| 醫療法 §82 | 仍適用（醫師個人責任） |
| 醫療法 §84 §85 | 廣告規範仍適用，再生醫療廣告**更嚴** |
| 藥事法 | 製劑條例為**藥事法的特別法** |
| 人體研究法、人體試驗管理辦法 | 試驗階段仍適用 |
| 個資法 §6 | 細胞 = 特種個資 |

## 七、PGY 三大自保提醒

### 1️⃣ 不要被當「人頭醫師」
- 診所老闆說「你只要掛名」 → 共同正犯
- 細胞療法執行 → 必須親自診察 + 知情同意 + 簽紀錄

### 2️⃣ 廣告小心審
- IG 發病人案例 → 即使匿名，可能違反 §39 + 個資法
- 「打了某某細胞療法皮膚變好」 → 即視同醫療廣告

### 3️⃣ 病人問「外面打的幹細胞會不會有效」
- 不需評論他院療效（避免醫療廣告）
- 衛教重點：「請確認該機構是否經衛福部核准」
- 核准名單：衛福部醫事司網站

## 八、罰則總覽

| 違法行為 | 法源 | 罰鍰 |
|---|---|---|
| 非醫療機構執行再生醫療 | 再生醫療法 §32 | 100–2,000 萬 |
| 醫師資格不符 | §33 | 50–500 萬 |
| 違反知情同意 | §34 | 20–100 萬 |
| 違法廣告 / 招募 | §39 | 50–200 萬 |
| 製劑未經查驗登記上市 | 製劑條例 §39 | 同藥事法（最高 2,000 萬以上） |

## 十、口袋提醒
> **「再生醫療不是普通醫療」 — 走錯機構/沒資格 = 醫師職業生涯終結級風險。**
> 醫美診所推銷你 moonlight 打幹細胞 → 立刻拒絕，2,000 萬不是嚇你的。
