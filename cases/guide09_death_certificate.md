# 防身指南 09｜死亡證明書／死產證明書 填寫地雷

## 來源
- 醫師法 §16、§17
- 刑法 §215（業務文書登載不實）、§213（公文書登載不實，不適用醫師）
- 刑事訴訟法 §218（檢察官相驗）
- 信心等級：高

---

## 一、白話結論

**死亡證明書是「業務上作成的文書」 → 寫錯/編造 = 刑法 §215，3 年以下徒刑。**
家屬施壓改死因 = **拒絕**，不是「人情」可以解的。

---

## 二、什麼時候你「不能」開死亡證明？

### 必須報請相驗的 6 種情境（醫師法 §16）

「醫師檢驗屍體或死產兒，如為**非病死或可疑為非病死**者，**應報請檢察機關依法相驗**。」

| 情境 | 處理 |
|---|---|
| 1. 自殺（吊死、跳樓、燒炭、藥物） | 報相驗，**不**自己開死亡證明 |
| 2. 他殺（傷害、打鬥、刀槍） | 報相驗 |
| 3. 意外（車禍、墜落、溺水、燙傷） | 報相驗 |
| 4. 死因不明 | 報相驗 |
| 5. 病房內可疑死亡（病情不符、突然死亡） | 報相驗 |
| 6. 醫療糾紛中的死亡 | 報相驗（保護自己） |

→ **報相驗 = 不是承認你有錯**，而是法律規定。

---

## 三、可以開的情境（病死且死因明確）

### 條件
1. **你有親自診察過**（醫師法 §17）
2. **死因明確且為自然病死**
3. **死亡發生在你的醫療照護期間**

### 標準格式（衛福部公版）
| 欄位 | 注意 |
|---|---|
| 直接死因（甲）| 最後導致死亡的原因（例：呼吸衰竭） |
| 先行原因（乙）| 引起甲的疾病（例：肺炎） |
| 先行原因（丙）| 引起乙的疾病（例：糖尿病） |
| 其他重要疾病 | 不直接致死但有貢獻 |
| 死亡方式 | 病死 / 非病死 |

⚠️ **不要寫「Cardiac arrest（心跳停止）」當主要死因** — 這是「死亡狀態」，不是死因。

## 四、家屬常見施壓 → 一律拒絕

| 家屬要求 | 違法之處 | 後果 |
|---|---|---|
| 「自殺寫成自然死，保險才賠」 | 刑法 §215 + 詐欺幫助 | 3 年徒刑 |
| 「車禍寫成中風」 | §215 + 妨害司法 | 3 年徒刑 |
| 「他殺寫成自然死」 | §215 + 湮滅證據（§165） | 3+5 年 |
| 「死亡時間提早 1 天」（保險生效） | §215 + 詐欺幫助 | 3 年 + 連帶賠償 |
| 「把抽菸/喝酒拿掉」 | 若不影響死因可酌情；影響因果則 §215 | — |

**SOP**：「對不起，這是法律規定我必須照實寫，否則我會被判刑。」

## 五、特殊情境

### 1️⃣ 病人在家死亡，家屬抱來醫院
- 你**沒**親自診察過 → 不能開（醫師法 §17）
- 報請當地衛生所或檢察官相驗
- 家屬怒 → 「我沒看過他的病歷無法判定，必須相驗」

### 2️⃣ 急診 DOA（到院前心跳停止）
- 看是否能查到病史 + 死因明確
- 不確定 → 報相驗
- 急救過後死亡 → 自己醫療照護期間 → 可開

### 3️⃣ 死產證明書
- 醫師法 §16 同樣適用
- 死產 ≥ 滿 20 週的胎兒
- 死因不明 / 可疑外力 → 報相驗

### 4️⃣ 安寧 / 在宅死亡
- 安寧 / 居家療護 + 末期 + 醫師有照護紀錄 → 可開
- 安寧緩和醫療條例 + 醫療法 §73-1（居家醫療）

### 5️⃣ 老人猝死（無病史）
- 沒查過病史的猝死 → 不要 assume「老死」
- 報相驗 → 確認非外力

---

## 六、相驗 ≠ 解剖

| 程序 | 誰執行 | 是否切開遺體 |
|---|---|---|
| 相驗 | 檢察官 + 法醫 | 否（外觀檢查） |
| 解剖 | 法醫 | 是（家屬可能不同意） |

→ 報相驗只是「請檢察官來看」，多數情況不會解剖。

---

## 七、開錯的後果

| 違法樣態 | 法源 | 刑度 |
|---|---|---|
| 故意改死因（配合家屬） | 刑法 §215 | 3 年以下 |
| 沒看過病人開證明 | 醫師法 §17 + 刑法 §215 | 行政罰 + 3 年以下 |
| 應相驗未相驗 | 醫師法 §16 + 刑法 §165 | 行政罰 + 5 年以下 |
| 受託代開（人情） | §215 + §17 | 同上 |

實務上：**密醫式代開** + **配合保險詐欺** 是兩大紅線。

## 八、引用法條
- 醫師法 §16（相驗報請義務）、§17（親自診察）
- 刑法 §215（業務文書登載不實）、§165（湮滅證據）
- 刑事訴訟法 §218（檢察官相驗）
- 安寧緩和醫療條例

> **「家屬會難過，但你會被關。」**
> 拒絕施壓不是無情 — 是保護你自己 + 病人最後的真相。
