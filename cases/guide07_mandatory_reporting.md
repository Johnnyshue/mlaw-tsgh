# 防身指南 07｜醫師通報義務一覽（沒通報 = 自己變被告）

## 來源
- 各別作用法（兒少法、家暴法、性侵害防治法、傳染病防治法、職災保險法、刑法 §165 等）
- 信心等級：高（條號以全國法規資料庫為準）

---

## 一、白話結論（30 秒）

**法定通報義務 = 法律強制，不通報自己有刑事/行政罰。**
共 12+ 種，**每種「通報窗口」「期限」「罰則」都不一樣**，背不起來就把這篇收進口袋。

## 三、最容易忘記/最容易漏的 5 種

### 1️⃣ 「家暴」不限身體傷害
- 言語威脅、控制、跟蹤、騷擾 = 都算
- 家暴法 §3 定義超廣，**老人配偶間的精神虐待也算**
- 病人哭訴丈夫罵她 → 你已經「知悉」 → 24h 通報

### 2️⃣ 「兒少疏忽」不需要傷
- 沒帶就醫 / 飲食不當 / 居住環境危險 = 疏忽
- 嬰兒搖晃症候群、顱骨骨折模式異常 → **強烈懷疑虐待**
- 兒少法 §53 列舉非常廣

### 3️⃣ 老人保護常被忽略
- 失能老人臥床長褥瘡、營養不良、用藥不順從 → 可能是被遺棄/虐待
- 老人福利法 §43 通報義務

### 4️⃣ 非病死必須報相驗
- 醫師法 §16：**「醫師檢驗屍體或死產兒，如為非病死或可疑為非病死者，應報請檢察機關依法相驗。」**
- 自己開死亡證明 = 偽造文書（刑法 §215）
- 案例：病人在病房打架後死亡，醫師寫「自然死亡」 → 刑事

### 5️⃣ 嚴重藥物不良反應
- Stevens-Johnson、TEN、agranulocytosis、肝衰竭 → **15 天內**通報
- 沒通報 = 行政罰 + 影響日後藥害救濟申請

## 四、實務 SOP（值班遇到時）

```
1. 確認懷疑 → 不需「100% 確定」才通報
2. 病歷紀錄客觀事實（不要寫主觀判斷）
   ✅「左眼周瘀青、形狀如手指印」
   ❌「應為家暴造成」
3. 通報窗口：113（暴力）/1957（老福）/院內社工
4. 院內社工會接手 + 協助通報
5. 病歷寫「已通報 OOO，案號 XXX」
6. 不要在病人/家屬面前說「我要通報你」
```

## 五、不通報的後果

| 情況 | 後果 |
|---|---|
| 兒虐沒通報 → 兒童後續死亡 | 行政罰 + 民事連帶 + 刑法 §276 過失致死 |
| 家暴沒通報 → 配偶被殺 | 行政罰 + 民事 |
| 法傳沒通報 → 群聚感染 | 9–45 萬罰鍰 + 影響執照 |
| 非病死寫病死 | 刑法 §215 業務文書登載不實（3 年以下）+ §165 湮滅證據 |

## 七、口袋提醒
> **看到任何「異常」 → 寧可通報 → 不通報自己反而變被告。**
> 通報後是社工/警察的事，醫師的責任就完成了。
