## 一、白話結論

**醫預法上路一年多了，實際感受：訴訟率有降，但「調解品質」差很多。**
最關鍵的兩件事：
1. **道歉條款**真的有用（§31）
2. **強制調解**有些縣市排程拖到 6 個月

## 二、醫預法的 3 大核心制度（半年實戰觀察）

### 1️⃣ 強制調解先行（§15、§16）— 訴訟前必經
- 民事訴訟前必須先調解
- **刑事偵查/審理停止** → 限制「以刑逼民」
- 實戰觀察：
  - 排程：醫學中心地區可能 2–6 個月
  - 調解委員：醫師、律師、社會公正人士（3 人合議）
  - 調解通知書 → 醫師**必須**參加（不參加可罰）

### 2️⃣ 道歉條款（§31）— 醫師護身符
> 「**調解過程中關於遺憾、道歉**、撫慰、所為之陳述或讓步，**除當事人均同意外**，不得於後續民、刑事訴訟、行政處分中採為證據或裁判基礎。」

實戰用法：
- ✅ 「對於這個結果我們很遺憾，我們陪伴家屬走過這段。」
- ✅ 「我們檢討過程式發現可改善的環節是 OOO，未來我們會調整。」
- ❌ 「我們確實有疏失，我承認過失。」（**承認過失不在保護範圍**）

### 3️⃣ 醫療事故關懷小組（§6）
- 重大事故（死亡、重傷殘）→ 醫療機構必須在 **72 小時內**組成關懷小組
- 醫師、護理師、社工、必要時律師
- 沒組成 → 罰 1–5 萬

## 三、半年來常見的「眉角」問題

### Q1：「我已經承認過失了，調解失敗會被當證據嗎？」
- **承認過失** → 不受 §31 保護 → 會被當證據
- 「遺憾、道歉」 → 受保護
- 區別關鍵：**有沒有自己承擔法律責任的意思表示**

### Q2：「我可以不去調解嗎？」
- 醫師個人收到通知 → 必須出席
- 不出席 → 可裁罰 + 視同放棄主張
- 實務：請醫院法務 + 個人保險指派律師陪同

### Q3：「調解成立後可以反悔嗎？」
- 調解書經法院核定 = **同民事確定判決**（§28）
- 不能再提同一事件民、刑事
- 反悔 = 違約金或強制執行

### Q4：「家屬告刑事我才知道，沒先走調解可以嗎？」
- 刑事告訴 → 檢察官會通知雙方先調解（§15）
- 調解期間刑事偵查停止
- 60 天內未成立 → 刑事偵查恢復

### Q5：「PGY 出事，主治和我都要去調解嗎？」
- 是的，被告/被申訴的醫師 + 醫療機構 + 受害方
- 三方代表都到場
- 院方法務通常會統一準備

## 四、生產事故救濟條例（婦產科獨立制度，2016）

醫預法之外，婦產科還有專屬制度：
- 適用：母嬰生產相關死亡 / 重度殘疾
- **無過失補償**：不需證明醫師過失
- 補償上限 200 萬
- 申請電話：02-2100-2092
- 不影響訴訟權，但實證 **降低 70% 訴訟率**

## 五、實戰 SOP（被通知參加調解時）

```
1. 第一時間通知個人保險公司 → 保險指派律師
2. 通知醫院法務 + 醫療糾紛處理小組
3. 蒐集所有證據：病歷、影像、護理紀錄、LINE 紀錄
4. 個人準備 timeline + 醫療常規依據（教科書、guideline）
5. 調解當天：
   - 不單獨見對方
   - 讓律師、院方法務當主談人
   - 醫師個人主要負責「醫學說明」+「表達遺憾」
   - 不對賠償金額個別承諾
6. 調解內容必書面紀錄 → 後續可能成為法院核定基礎
7. 不成立 → 進入訴訟，調解陳述（除「遺憾/道歉」外）可採為證據
```

## 六、新法上路一年的數據觀察

| 指標 | 上路前（2023）| 上路後（2024–25 推估） |
|---|---|---|
| 民事案件數 | 約 500 件/年 | 略降，但調解案件大增 |
| 調解成立率 | – | 約 40–50% |
| 平均調解時間 | – | 2–6 個月 |
| 醫師「遺憾道歉」使用率 | 低 | **顯著上升** |
