# 各科地雷 07｜精神科（2024 精衛法修法後新規則）

## 來源
- 精神衛生法（2022/12/14 三讀，**2024/12/14 全面施行**）
- 全文：https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020030
- 信心等級：高（強制住院新制條文已生效，但部分施行細則仍演化）

---

## 一、白話結論

**「強制住院」從『行政程序』升級為『法院裁定』 — 醫師責任增、文書負擔增。**
2024 新法上路後，PGY 在精神科必須懂 4 個關鍵流程：**緊急安置 → 強制鑑定 → 法院聲請 → 強制治療**。

---

## 二、新法 vs 舊法（強制住院程序）

| 階段 | 舊法（–2024/12/14）| 新法（2024/12/14 起） |
|---|---|---|
| 強制住院決定者 | 強制住院審查會（行政） | **法院裁定**（司法） |
| 法院組成 | – | **參審制**（法官 + 精神科專科 + 病權團體代表） |
| 緊急安置期 | 5 天 | **7 天** |
| 強制鑑定 | 2 天 | **3 天** |
| 強制住院上限 | 60 天 | **60 天**（延長 1 次，60 天以內） |
| 病權團體 | 無參與 | **必須參與審理** |

---

## 三、強制住院流程（新法 SOP）

```
1. 病人符合「嚴重病人」+ 自傷傷人之虞
   ↓
2. 警消／醫療人員 → 緊急安置（最多 7 天）
   ↓
3. 兩位以上指定精神科專科醫師鑑定（緊急安置次日起 3 天內）
   ↓
4. 鑑定有強制住院必要 → 醫療機構向法院聲請
   ↓
5. 法院參審制裁定（法官 + 精神科 + 病權代表）
   ↓
6. 強制住院 ≤ 60 天
   ↓
7. 需延長 → 屆滿 14 天前向法院聲請（限 1 次，再 ≤ 60 天）
```

---

## 四、PGY 5 大地雷

### 1️⃣ 緊急安置「逾期未鑑定」
- 緊急安置 7 天 + **3 天內必須完成強制鑑定**
- 沒鑑定 / 文書漏 → **違法拘禁** → 民事 + 刑法 §302
- 案例：某醫院安置 8 天才鑑定 → 病人提告勝訴

### 2️⃣ 沒兩位專科醫師鑑定就硬留人
- §41：必須兩位**指定精神科專科醫師**鑑定
- PGY 不是專科醫師 → 不能單獨鑑定
- 缺一個 → 整個強制無效

### 3️⃣ 保護性約束 SOP 不熟
- §37 保護性約束：**最後手段**、有時限、**每 24 小時必須評估**
- 病歷必寫：理由、時間、評估、解除
- 沒評估就持續綁 → 民事侵權 + 刑事妨害自由

### 4️⃣ 自殺通報 + 病歷紀錄
- 法定通報：**自殺企圖 / 自傷者** → 衛生局通報（精神衛生法 §51）
- 病歷必寫風險評估（C-SSRS / 簡易評估）
- 自殺出院後死亡 → 你病歷沒寫評估 → 民事敗訴大宗
- 案例：多起精神科出院後自殺案 → 醫師被判賠

### 5️⃣ 強制治療 ≠ 強制服藥
- 法院裁定的是「**強制住院**」
- 院內仍需病人**同意接受治療**（除非更高層次的緊急介入）
- 病人拒藥 → 不能硬給（除非生命威脅）
- 強迫給藥 → 違反醫療倫理 + 民事

## 五、社區治療 vs 強制住院

| 制度 | 適用 | 決定者 |
|---|---|---|
| 強制住院 | 嚴重精神病人有自傷傷人之虞 | **法院** |
| 強制社區治療 | 拒絕治療但不需住院的嚴重病人 | **強制社區治療審查會** |

→ 新法將「強制住院」司法化，「強制社區治療」仍由審查會。

## 六、其他常見訴訟態樣

| 案例類型 | 法律風險 |
|---|---|
| 自殺出院後死亡 | 民事過失（評估不足） |
| 約束病人受傷 | 民事 + 妨害自由 |
| 強迫送醫 | 違反 §41、§42 程序 |
| 病歷外洩（明星就診） | 個資法 §41（5 年以下） |
| 假病歷便利保險理賠 | 刑法 §215 |
| 鑑定報告偏頗（家事/刑事） | 偽證罪、行政懲戒 |

## 七、與其他法律的關係

| 法律 | 銜接點 |
|---|---|
| 醫療法 | 強制住院仍適用 §82 醫療責任 |
| 個資法 §6 | 精神病歷 = 特種個資 + 高度敏感 |
| 病主法 | 精神疾病不在 5 大臨床條件，但末期共病可適用 |
| 安寧緩和條例 | 共病末期病人可適用 |
| 民法（行為能力） | 嚴重病人可能涉及監護宣告 |

---

## 八、PGY 自保 4 條

1. **病歷詳實**：自殺風險評估、約束評估、藥物反應
2. **流程不能跳**：兩位專科鑑定 / 法院聲請 / 24 小時評估
3. **「拒絕治療」要記錄**：病人/家屬拒絕、衛教內容、簽 AAD
4. **個資保密最重**：精神病歷外洩 = 5 年以下
