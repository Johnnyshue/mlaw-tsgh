# 法律分析 09｜個資法 §6 與社群媒體（醫師最易踩、最重的雷）

## 來源
- 個人資料保護法（PDPA）
- 全文：https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=I0050021
- 醫師法 §23（保密義務）
- 信心等級：高

---

## 一、白話結論

**「病歷資料 = 特種個資 = 最高刑度 5 年。」**
即使匿名、即使是「教學分享」、即使是私人 IG 限動 — 只要可識別 = 違法。

---

## 二、為什麼醫療資料這麼敏感？

個資法 §6 把這些列為「**特種個人資料**」：
- 病歷
- 醫療
- 基因
- 性生活
- 健康檢查
- 犯罪前科

→ 原則上 **不得蒐集、處理、利用**。
→ 例外：法定情形（治療、保險申請、法律規定、公共利益）+ **書面同意**。

---

## 三、刑度與罰則總覽

| 違法樣態 | 法源 | 處罰 |
|---|---|---|
| 違反 §6 處理特種個資（例：未授權外洩病歷） | §41 | **5 年以下徒刑**、得併科 100 萬以下罰金 |
| 一般個資非法處理 | §41 | 5 年以下徒刑、100 萬以下罰金 |
| 民事損害賠償（每件 500–20,000 元，總額上限 2 億） | §28 | 民事連帶 |
| 行政罰鍰 | §47–§50 | 5–50 萬 |
| 醫師法保密義務違反 | 醫師法 §23 | 行政罰 + 醫師法處分 |

## 四、5 大社群媒體陷阱

### 1️⃣ IG 限動發 case
- ⛔ 即使匿名「今天遇到一個 90 歲奶奶 GTC seizure」 → 同一病房同事認得出來 = 可識別
- 同時違反個資法 §6 + §41 + 醫師法 §23

### 2️⃣ FB 群組「我們醫院今天有個有趣 case」
- ⛔ 即使是「私人群組」 — 群組仍是處理行為
- 一旦截圖外流 → 來源就是你
- 法院實務：**「私人群組」不能阻卻違法**

### 3️⃣ LINE 工作群組分享 X 光片
- ✅ 醫療目的（會診、教學）+ 群組成員都是醫療人員 → 一般合法
- ⛔ 截圖到一般 LINE / 私人聊天 → 可能違法
- ⛔ 把病人 X 光當「梗圖」轉發 → 違法

### 4️⃣ 部落格／IG 教學文
- 「分享一個有趣案例」 → 必須完全去識別化：
  - 拿掉時間、地點、年齡（或粗略化）
  - 拿掉特殊診斷（罕病可能反向識別）
  - 不貼影像（即使去個資，醫院 PACS 浮水印仍可識別）
- 最安全：**用模擬病人**（fictional case）

### 5️⃣ 匿名向同事抱怨
- 「我同事 OOO 太爛」 → 違法（誹謗 §310）
- 「今天 ICU 那個 30 歲外傷腦死」 → 即使沒名字，仍可能識別 → 違法

## 五、什麼情況可以「合法」處理病人資料？

依個資法 §6 但書 + §16，必須符合**任一**：
1. **法律明文規定**（健保申報、傳染病通報、強制鑑定）
2. **公務機關執行法定職務**
3. **學術研究**（已去識別化 + 倫委會審查）
4. **書面同意**（病人簽署、明確、自由意志）
5. **公共利益必要**（疫情、災害）
6. **醫療治療必要**（會診、轉院摘要）

→ 教學發表 / 期刊 case report → **必須得書面同意 + 倫委會**。

## 六、實戰：醫師最常被告的 4 種情境

### 情境 1：病人發現自己病歷在 FB 上
- 路徑：醫師 → 同事 LINE → 截圖外流
- 病人提告 → 民事每人 500–20,000 元 + §41 刑事
- 醫院連帶責任 + 涉案醫師重責

### 情境 2：診所員工外洩名人就診紀錄給媒體
- 員工 + 院長皆罰
- §47 行政 + §41 刑事
- 院長未盡管理義務 → 連帶責任

### 情境 3：研究論文 case report 沒拿同意書
- 期刊撤稿 + 倫委會處分
- 醫院違反 §6 → 行政罰
- 影響升等、申請計畫資格

### 情境 4：實習醫師上 IG 限動「今天 cardiac arrest 救回來爽」
- 病人家屬看到提告
- 學校紀律 + 醫師法處分（停業）+ §41 刑事
- 已有實際案例（醫院實習生被停權）

---

## 七、安全 SOP（教學/研究/分享前 5 個檢查點）

```
□ 1. 是否可識別？（病人 + 家屬 + 同事 + 病房）
□ 2. 是否有書面同意？（特種個資原則上要）
□ 3. 是否經倫委會審查？（研究必經）
□ 4. 是否拿掉影像浮水印 / metadata？
□ 5. 散布範圍是否最小化？
```

5 個全 ✓ 才能發。任一 ✗ → 不發。

---

## 八、與健保詐欺 / 醫療廣告法的關係

- 健保資料 = 個資 → 同樣受保護
- 健保詐欺案件中的個資使用 → §47 + §81 健保法 + 詐欺罪
- 醫療廣告 → 不可使用病人個資、見證、Before/After（醫療法 §86）
