# 各科地雷 08｜神經科（tPA 黃金時效 + SAH 漏診）

## 來源
- 醫療法 §82
- 各國中風指引（AHA/ASA、台灣腦中風學會）
- 司法實務：多起 tPA 延誤、SAH 漏診賠償判決
- 信心等級：高（時效黃金期已成醫療常規共識）

---

## 一、白話結論

**神經科兩大訴訟：「tPA 黃金期錯過」 + 「SAH 被當作偏頭痛」。**
病歷上**沒寫評估時間、沒寫禁忌症評估、沒寫家屬告知** = 鑑定報告幾乎必負責。

---

## 二、5 大訴訟高風險情境

### 1️⃣ tPA / EVT 黃金時效
| 項目 | 規範 |
|---|---|
| Door-to-needle | ≤ 60 分（標竿 ≤ 45） |
| Door-to-CT | ≤ 25 分 |
| tPA 治療窗 | 4.5 小時內（特殊條件 9 小時） |
| EVT（取栓）治療窗 | 大血管 LVO 6 小時，特殊 16–24 小時 |

法律重點：
- **延誤的每分鐘** → 病歷必須有「為什麼延誤」的合理理由
- 沒理由的延誤 = 過失推定
- 病歷沒寫 NIHSS 評估 → 你說「我評估過了」鑑定不採

### 2️⃣ SAH 漏診（最大宗訴訟）
- 25–50% 的 SAH 一開始被誤診（國外文獻）
- Worst headache of life / thunderclap → **必須**懷疑 SAH
- CT 陰性 ≠ 排除 SAH（敏感度隨時間下降）
  - < 6 小時：99%
  - 6–24 小時：93%
  - > 24 小時：明顯下降 → **LP**
- 沒做 CT、沒考慮 LP → 漏診 → 過失

### 3️⃣ tPA 禁忌症漏看 → 出血
- 血壓 > 185/110 沒控制
- INR > 1.7、近期手術、近期出血、aneurysm
- 漏看 → 病人出血死亡 → 民事 + 可能刑事
- 病歷必寫：禁忌症 checklist 全部 ✓

### 4️⃣ 中風病人未告知 tPA / EVT 選項
- 知情同意義務（醫療法 §63 §81）
- 病人 / 家屬不知有此選項 → 拒絕後死亡 → 民事敗訴
- 病歷必寫：「已告知 tPA 風險效益、家屬選擇 OOO」

### 5️⃣ 失能病人離院後跌倒 / 二次中風
- 出院教育（§81）：跌倒風險、復健、用藥順從
- 沒衛教 → 二次中風 → 因果關係
- 病歷必寫：出院衛教內容、複診時間

## 三、神經科常見鑑定爭點

| 爭點 | 鑑定重點 |
|---|---|
| 是否「該做 CT」 | 病人主訴、神經學檢查、危險因子 |
| 是否「該打 tPA」 | 時間窗、NIHSS、禁忌症 |
| 是否「該轉 EVT 中心」 | LASS / 大血管中風判斷、轉院能力 |
| 是否「該做 LP」 | CT 陰性、SAH 懷疑度、時間 |
| 是否「告知充分」 | 同意書、衛教紀錄 |

---

## 四、PGY 神經科自保 5 條

### 1️⃣ NIHSS 永遠寫進病歷
- 第一次 NIHSS、tPA 前、tPA 後 2 / 24 小時
- 沒寫 = 鑑定無依據 = 你不利

### 2️⃣ Worst headache 視為 SAH 直到證明非 SAH
- 「最痛」「突發」「雷擊樣」 → CT + 必要時 LP
- 病歷不要寫「Tension headache, like before」就讓人走

### 3️⃣ 中風 stroke alert 啟動所有時間點記錄
- LKW（last known well）、door 時間、CT 時間、tPA 開始
- 鑑定看時間軸

### 4️⃣ 高齡跌倒外傷必查抗凝藥
- Aspirin / clopidogrel / NOAC / warfarin → CT 閾值更低
- 即使「沒症狀」，年齡 > 65 + 抗凝藥 → 強烈建議 CT

### 5️⃣ 與家屬溝通必書面
- tPA 風險：6% 顱內出血、2–3% 致死
- 拒絕者簽拒絕書 + 病歷紀錄

---

## 五、tPA 流程速查 SOP

```
1. 確定 LKW（最後正常時間）
2. NIHSS 評估
3. CT 排除出血
4. 禁忌症 checklist
5. BP < 185/110（必要時降壓）
6. 體重 → 劑量（0.9 mg/kg, 最大 90 mg, 10% bolus）
7. 家屬告知 + 同意書
8. 治療後 24h ICU 監測（無 antiplatelet/anticoag）
```
