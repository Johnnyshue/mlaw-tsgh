# 各科地雷 10｜牙科（自費糾紛大宗 + Case15 / Case16 系統論述）

## 來源
- 醫療法 §63、§81、§82
- 牙醫師法
- 案例佐證：Case 15（植牙傷鄰牙）、Case 16（口腔潰瘍延誤轉診）
- 信心等級：高

---

## 一、白話結論

**牙科是「自費糾紛 + 損害結果明確」的高訴訟科別。**
最大兩類：(1) **植牙併發症**、(2) **延誤轉診（口腔癌）**。

---

## 二、5 大訴訟高風險情境

### 1️⃣ 植牙併發症（傷鄰牙、神經、副鼻竇）
（參 Case 15）
- 鑽孔角度錯 → 傷鄰牙、下齒槽神經
- 上顎植牙穿入副鼻竇 → 副鼻竇炎
- 損害結果**明確可量化** → 民事易勝
- **同意書必須詳列風險**（神經、感染、植體失敗、骨整合不全）

### 2️⃣ 延誤轉診（口腔潰瘍 → 口腔癌）
（參 Case 16）
- 潰瘍 > 2 週 → 必須懷疑癌症
- **病歷必寫**：建議切片 / 轉診醫學中心、病人決定
- 沒建議轉診 → 延誤診斷 → 民事 + 可能刑事
- 牙醫師法 §15、醫療法 §73 轉診義務

### 3️⃣ 拔牙併發症（鎮靜麻醉死亡）
- 鎮靜麻醉 → 必須有監測 + 急救設備
- 急救設備不足 / 沒登記為麻醉場所 → 違規
- 案例：診所拔智齒鎮靜後死亡 → 醫師刑事 + 民事

### 4️⃣ 矯正治療糾紛
- 治療前未充分告知療程、結果範圍、費用
- 結果不如預期 → 退費糾紛 + 民事
- 同意書必寫：療程時間、費用、可能結果範圍、無法保證

### 5️⃣ 美容牙科（瓷牙、貼片）
- 「Before/After」廣告 → 違反醫療法 §86 醫療廣告限制
- 自費高額糾紛
- 治療不可逆（磨牙）→ 民事

---

## 三、自費醫療糾紛特性

| 自費醫療 | 健保醫療 |
|---|---|
| 病人付了錢 → 期待值高 | 期待值較低 |
| 易主張「契約違反」（民法 §227） | 主要走醫療過失 |
| **舉證責任有時較重於醫師**（消保法、醫療法 §63） | 一般醫療過失原則 |
| 賠償常含**已支付費用 + 損害** | 主要為損害 |

---

## 四、牙科同意書必有 7 項（Case 19 示範）

1. 治療項目（具體名稱）
2. 預期效果
3. 可能風險（量化機率）
4. 替代方案 + 替代方案的優缺點
5. 拒絕治療的後果
6. 費用明細（自費）
7. **病人簽名 + 醫師簽名 + 日期 + 時間**

---

## 五、轉診時機（牙醫師必懂）

| 情境 | 轉診對象 |
|---|---|
| 口腔潰瘍 > 2 週 | 口腔顎面外科 / 醫學中心 |
| 不明腫塊 | 同上 |
| 顳顎關節持續疼痛 | 顳顎關節科 / 復健 |
| 嚴重牙周破壞 | 牙周專科 |
| 顎骨骨折 | 口腔顎面外科急診 |

牙醫師法 §15：「牙醫師對於診治之病人，得交付處方書、醫囑書或證明書；如為診治需要，並得依其情況，協助病人轉診**或為其他必要之指示**。」

---

## 六、PGY / 牙科醫師自保 5 條

1. **同意書要量化**：「神經損傷率 < 1%」比「可能傷神經」好
2. **植牙前必拍 CBCT**：沒拍 = 鑑定推定醫療常規違反
3. **延誤轉診是大宗**：潰瘍 > 2 週、不明腫塊 → 寫衛教 + 建議轉診
4. **鎮靜麻醉需登記**：診所自行麻醉須符合規範
5. **自費糾紛走民事**：病人不滿 → 院方統一處理 + 保留書面

## 九、口袋提醒
> **「牙科糾紛 = 自費 + 損害可見 + 病人付了錢」 = 三高訴訟風險。**
> 同意書量化、延誤轉診不能省、鎮靜麻醉合規 — 三條紅線記住即可。
