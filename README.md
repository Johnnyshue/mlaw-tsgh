## 防造假原則

- **法條**：每條都從全國法規資料庫即時爬取，附原始連結
- **章節**：所有條號、條文內容都來自爬蟲資料
- **判決**：因司法院反爬，只放搜尋指引與裁判字號連結，不編造判決細節
- **統計**：引用醫改會與衛福部公開資料，附原始連結
