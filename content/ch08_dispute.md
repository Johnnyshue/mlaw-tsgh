# Ch8 醫糾發生那一刻：道歉條款與爭議處理法

## 醫療事故預防及爭議處理法（民 113 全面施行）

民國 111 年 6 月制定公布，民國 113 年 1 月 1 日全面施行。**這是醫師最大的好消息**。

來源：[全國法規資料庫](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020227)

## 三大重點

### 重點 1：道歉條款（§7）

> 進行說明、溝通、提供協助及關懷服務過程中，醫療機構、醫療事故關懷小組、專業人員、醫事人員或其代理人所為**遺憾、道歉、讓步**或其他為緩和醫病緊張關係所為之陳述，**除醫療爭議當事人均同意外，不得於本案訴訟採為證據或裁判基礎**，亦不得採為相關行政處分之基礎。

→ **白話**：你說「對不起，這個結果不是我們希望的」**不會在法庭上被當證據用**。

> ⚠️ 注意：是「遺憾/道歉/讓步」這類「**緩和緊張關係**」的話，不包括承認過失或編造事實。

### 重點 2：醫療事故關懷小組（§6）

> 醫療事故發生之翌日起 5 個工作日內，向病人、家屬或其代理人說明、溝通，並提供協助及關懷服務。

→ **醫療機構義務**：99 床以下醫院/診所可委外或指定專人。

### 重點 3：強制調解前置（§13、§14、§15）

> 民事訴訟提起前，**應先經調解**。

→ **白話**：病人/家屬要告民事，必須先經過縣市衛生局調解會調解。

## 醫療事故發生後的「黃金 72 小時 SOP」

```
[第 1 小時]
  - 立即停止可能持續造成損害的處置
  - 通報主治 / 科主任 / 院方主管
  - 不要單獨向家屬說明

[第 1-24 小時]
  - 院方關懷小組介入
  - 病歷補完整（依醫療法 §68 規定的合法格式）
  - 不要修改、塗銷既有病歷
  - 與護理紀錄對齊時間軸

[24-72 小時]
  - 院方關懷小組對家屬說明
  - 你的角色：陪同、傾聽、表達遺憾（適用 §7）
  - 不單獨對家屬承認過失或承諾賠償

[72 小時後]
  - 配合院方法務、保險公司
  - 必要時聯繫醫師公會 / 個人保險
```

## 醫師遇到醫糾的「五不」

| # | 不要做 | 為什麼 |
|---|---|---|
| 1 | **不要單獨見家屬** | 任何承認過失的話都會被截圖 / 錄音 |
| 2 | **不要修改病歷** | 違反 §68，刑事偽造文書 |
| 3 | **不要私下和解** | 沒走完院方流程，個人和解可能無保險覆蓋 |
| 4 | **不要刪 line / 通訊紀錄** | 這是證據，刪了反而被告湮滅證據 |
| 5 | **不要在社群發文** | 即使匿名，被肉搜後加重判決 |

## 醫師遇到醫糾的「五要」

| # | 要做 | 法源 / 出處 |
|---|---|---|
| 1 | **要立刻通報院方** | 醫療法 §107（重大事故通報） |
| 2 | **要表達遺憾** | 醫療事故預防及爭議處理法 §7（不會被採證） |
| 3 | **要保留所有紀錄** | 含 EKG、影像、護理紀錄、會診紀錄 |
| 4 | **要尋求法律諮詢** | 醫師公會、個人法律顧問、醫責險公司 |
| 5 | **要記得個人保險** | 醫師責任險覆蓋訴訟費 + 賠償 |

## 醫師責任險（個人保險）

> 不是只靠醫院的保險，**個人責任險**是必備的。

| 項目 | 說明 |
|---|---|
| 保額 | 多在 500 萬至 5000 萬 |
| 保費 | 依科別差異大（外科、婦產科、急診保費較高）|
| 覆蓋 | 訴訟費 + 賠償金 + 律師費 |
| 出險 | 案件成立時通知保險公司 |

## 衛生局調解流程

```
病人/家屬申請調解
  ↓
衛生局通知雙方
  ↓
調解會議（由律師、醫師、社會公正人士組成）
  ↓
調解成立 → 簽和解書 → 訴訟終結
  ↓
調解不成立 → 出具「調解不成立證明」 → 才能提起民事訴訟
```

## 真實統計（醫改會公開資料）

醫改會歷年資料顯示：
- 衛生局調處案件數**民事 >> 刑事**
- 多數爭議在**院內申訴 + 調解**階段就解決
- 進入司法程序的案件**定罪率不高**（學界引用約 5-10% 區間，依年份與來源不同）

來源：[醫改會 醫療爭議統計](https://www.thrf.org.tw/dispute/108)

## 結論：醫糾發生時「三心二意」

- **三心**：耐心（聽家屬講）、誠心（表達遺憾）、決心（不私下和解）
- **二意**：通報院方意願明確、保險通知意識立刻
