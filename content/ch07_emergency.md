# Ch7 急診：最容易被告的科別

## 為什麼急診最常被告

| 結構性因素 | 法律後果 |
|---|---|
| 醫病關係短 | 信任低 → 一有狀況直接告 |
| 病人多重診斷 | 任何一個漏診都是攻擊點 |
| 病情變化快 | 「漏看」很常見，鑑定容易抓到 |
| 24 小時運作 | 値班疲勞 → 錯誤率高 |
| 醫療資源有限 | 設備未到位的「漏」也算 |

> 文獻一致顯示：**急診是醫療糾紛最高風險科別之一**（與外科、婦產科並列前三）。

## 急診五大死亡 / 漏診紅旗（最容易被告的情境）

### 🚨 紅旗 1：胸痛 → 漏掉 STEMI / aortic dissection / PE

**法官會問**：
- 有沒有做 EKG？多久內做？
- 有沒有抽 TnI？
- 有沒有 D-dimer？
- 有沒有問家族史 / 抽菸史？

**防身**：
- 30 歲以上胸痛 → EKG 必做（10 分鐘內）
- 心電圖 normal 不代表沒事 → TnI 0h + 3h 至少兩次
- 撕裂痛/背痛 → 想 dissection（D-dimer + CT）
- 暈厥 + 喘 + tachycardia → 想 PE（Wells / D-dimer）

### 🚨 紅旗 2：頭痛 → 漏掉 SAH / meningitis / 腦中風

**法官會問**：
- 有沒有做神經學檢查？
- 突發 thunderclap headache 有沒有 CT？
- 發燒 + 頭痛 + 頸僵硬 → 有沒有想 meningitis？

**防身**：
- 「This is the worst headache of my life」 → 立刻 CT，CT 陰再考慮 LP
- 50 歲以上新發頭痛 → 警戒
- 神經學檢查每次都要做（含瞳孔、四肢力氣、cerebellar）

### 🚨 紅旗 3：腹痛 → 漏掉 appendicitis / mesenteric ischemia / AAA

**法官會問**：
- 「pain out of proportion」有沒有想到 mesenteric ischemia？
- 老人腹痛有沒有想到 AAA？
- 闌尾炎延誤超過 24 小時穿孔，你的記錄怎麼寫？

**防身**：
- 老人腹痛 → 永遠做 CT
- pain disproportionate to exam → mesenteric ischemia red flag
- 婦女育齡 → 加 β-hCG（外子宮孕）

### 🚨 紅旗 4：頭部外傷 → 漏掉 ICH / EDH / SDH

**法官會問**：
- 有沒有做 GCS？多久重複一次？
- 抗凝血劑使用者有沒有 CT？
- 頭部外傷後**回家**的衛教有寫嗎？

**防身**：
- Canadian CT Head Rule / NICE Guideline 套用
- 抗凝者頭部外傷 = CT，不要省
- 出院衛教：24-48 小時觀察項目（嗜睡、嘔吐、頭痛加劇）→ 病歷寫**已衛教**

### 🚨 紅旗 5：發燒 → 漏掉 sepsis / meningitis（兒童）

**法官會問**：
- 有沒有 lactate？有沒有評估 qSOFA / SIRS？
- 抗生素多久內給？(Surviving Sepsis：1 小時內)
- 兒童 fever + irritability + bulging fontanelle 有沒有想 meningitis？

**防身**：
- Sepsis bundle 1 小時內：lactate / blood culture / 廣效抗生素 / fluid 30 mL/kg
- 嬰幼兒發燒 → 看門檻（< 3 個月 fever 直接 admit）

## 緊急醫療救護法的相關義務

> **緊急醫療救護法 §39**
>
> 急救責任醫院應依其指定之類別，對非預約之緊急傷病患，立即依其醫療能力提供適當之救治。

來源：[緊急醫療救護法](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020045)

→ **急診不能拒收**，但「能力範圍內」+ 「適當救治」是條件。能力之外要立即轉診（醫師法 §22）。

## 「轉診」的法律意義（醫師法 §11、§22）

> **醫師法 §22**
>
> 醫師…遇有疑難症候，認為應由其他醫院（所）醫師會診，為患者之健康應協助病人轉診之。

→ 不轉診 / 延誤轉診是急診常見敗訴原因。

## 「離院前衛教」的證據力

急診常見訴訟模式：

```
病人來急診 → 醫師說沒事 → 出院 → 隔天惡化死亡 → 家屬告
```

→ 醫師抗辯關鍵：**離院前衛教**有沒有寫進病歷？

**離院衛教必寫項目**：
- 何時回診（具體時間）
- 哪些症狀要立刻回急診（紅旗症狀）
- 衛教是否口頭 + 書面雙管
- 是誰接受衛教（病人本人 / 家屬）

> 病歷寫「**已向病人及配偶口頭及書面說明：若出現 X、Y、Z 立刻回急診**」是急診醫師的最強護身符。

## 急診五大「不得讓他走」

> 即使病人吵著要走，要有自動出院（AAD）程序 + 詳細病歷，否則出事還是會被告。

1. **胸痛 + 不正常 EKG / TnI**
2. **腹痛 + 急腹症兆候**（peritoneal sign）
3. **GCS < 15 / 神經學異常**
4. **Vital sign 不穩**（BP < 90, HR > 120, SpO2 < 92%）
5. **兒童 < 3 個月 fever**

→ 病人堅持出院 → 簽 AAD（自動出院）+ **病歷詳實記錄你的勸告 + 風險告知**。

## 結論：急診醫師的「五個自我提問」

每結束一個 case 前問自己：

1. 我的 SOAP 有沒有寫完整？
2. 紅旗症狀我有沒有評估到？
3. 該照的影像有沒有照？
4. 該抽的血有沒有抽？
5. **離院衛教**有沒有寫進病歷？

任一回答「沒有」 → 補上去。
