# Ch5 知情同意：怎麼簽才不會被翻盤

## 條文：醫師告知 4 大來源

### 醫療法 §63（手術同意書）
> 醫療機構實施手術，應向病人或其法定代理人、配偶、親屬或關係人說明手術原因、手術成功率或可能發生之併發症及危險，並經其同意，**簽具手術同意書及麻醉同意書**，始得為之。但情況緊急者，不在此限。

### 醫療法 §64（侵入性檢查）
> 醫療機構實施中央主管機關規定之**侵入性檢查或治療**，應向病人或其法定代理人、配偶、親屬或關係人說明，並經其同意，**簽具同意書**後，始得為之。但情況緊急者，不在此限。

### 醫療法 §81（一般告知義務）
> 醫療機構診治病人時，應向病人或其法定代理人、配偶、親屬或關係人**告知其病情、治療方針、處置、用藥、預後情形及可能之不良反應**。

### 醫師法 §12-1
> 醫師診治病人時，應向病人或其家屬告知其病情、治療方針、處置、用藥、預後情形及可能之不良反應。

來源：[醫療法](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020021)、[醫師法](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020001)

## 法官會問的 5 個告知要素（5W）

| W | 法官的問題 | 病歷該寫什麼 |
|---|---|---|
| **What** | 你告知了什麼？ | 病情 / 治療方針 / 替代方案 / 不治療的後果 |
| **Why** | 為什麼這樣建議？ | 臨床判斷 + guideline 根據 |
| **Risk** | 主要風險是什麼？ | 併發症發生率（如 cath 出血 1-3%）|
| **Who** | 對誰說？ | 病人本人 + 法定代理人 / 配偶 / 親屬 |
| **When** | 什麼時候說？ | 手術前**充分時間**（不是進手術房才簽）|

## 簽同意書的 7 個地雷

| # | 地雷 | 法官怎麼看 |
|---|---|---|
| 1 | **進手術房才簽** | 沒有「充分思考時間」= 同意可能無效 |
| 2 | **只簽不講** | 書面 + 口頭都要，缺一不可 |
| 3 | **未說明替代方案** | 病人若知道有別的選擇可能不會選這個 |
| 4 | **未說明「不治療的後果」** | 「無治療選項 (no treatment)」也是選項之一 |
| 5 | **代簽** | 病人有意識能力 → 必須本人簽；無能力 → 法定順位 |
| 6 | **空白同意書** | 同意書要列出**該次**手術的具體資訊 |
| 7 | **同意書 vs 病歷紀錄不一致** | 兩者要相符；病歷補強同意書內容 |

## 「家屬簽名順位」(醫療法 §63 II)

當病人**未成年或無法親自簽具**時：

```
法定代理人 → 配偶 → 親屬 → 關係人
```

> 「親屬」指**有事實關係**的家屬（不一定要直系），實務上以同住或最常照顧者為主。

## 知情同意 ≠ 萬靈丹

簽了同意書**不代表免責**：

- 同意書只能「免除已告知之風險」
- **過失**（手術切錯部位、藥開錯）= 同意書救不了你
- **未告知的風險**發生 = 仍有民事賠償可能

## 兩個特殊情境

### A. 病人意識不清，無家屬在場（急救）
- 醫療法 §63 但書「**情況緊急者不在此限**」
- 醫師法 §21「對危急病人不得無故拖延」
- → 先救命，事後補簽 + 詳實病歷紀錄

### B. 病人本人拒絕簽，但家屬同意（或反之）
- 病人**有能力時**：以病人本人意願為準
- 病人**無能力時**：依法定順位（配偶 > 親屬）
- 衝突時：建議由**社工 + 倫理委員會**介入，不要單一醫師決定

## 病人自主權利法 §5 §6（針對 5 種特定臨床條件）

> 病人對於病情、醫療選項及各選項之可能成效與風險預後，有知情之權利。

→ 病人自主權利法針對「**末期病人 / 不可逆昏迷 / 永久植物人 / 極重度失智 / 經中央主管機關公告之疾病**」這 5 種情況，有「**預立醫療決定（AD）**」的特別規範。詳見：[病人自主權利法](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0020189)。

## 結論：知情同意「五字訣」

1. **早**：手術前**至少數小時**，最好前一天
2. **講**：口頭充分說明（不是只簽字）
3. **聽**：給病人/家屬問問題的時間
4. **寫**：病歷紀錄當下溝通內容
5. **簽**：同意書 + 病歷紀錄都要有
